# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.<br>
&nbsp;<br>
Action Steps:</p>
<p>1. &nbsp; &nbsp;Click the Windows Search Bar located at the bottom-left corner of the Windows desktop<br>
2. &nbsp; &nbsp;Enter 'Environment' in the Search bar<br>
3. &nbsp; &nbsp;Press the Enter key.<br>
4. &nbsp; &nbsp;Verify that the 'System Properties' dialog opens<br>
5. &nbsp; &nbsp;Click the 'Environment Variables' button in the 'System Properties' dialog<br>
6. &nbsp; &nbsp;Verify that the 'Environment Variables' dialog box opens.&nbsp;<br>
7. &nbsp; &nbsp;Click the 'New...' button under the 'System Variables' section<br>
8. &nbsp; &nbsp;Verify that 'New System Variable' dialog opens.&nbsp;<br>
9. &nbsp; &nbsp;Enter 'FEEDBACK' in the Variable name field<br>
10. &nbsp; &nbsp;Enter 'FALSE' in the Variable value field&nbsp;<br>
11. &nbsp; &nbsp;Click the OK button.<br>
12. &nbsp; &nbsp;Verify that the 'New System variable' dialog closes.&nbsp;<br>
13. &nbsp; &nbsp;Verify that the newly entered Variable and Value are updated under the System Variables section<br>
14. &nbsp; &nbsp;Click the 'OK' button in the 'Environment Variables' dialog<br>
15. &nbsp; &nbsp;Verify that the 'Environment Variables' dialog closes<br>
16. &nbsp; &nbsp;Click the 'OK' button in the 'System Properties' dialog<br>
17. &nbsp; &nbsp;Verify that the 'System Properties' dialog closes<br>
18. &nbsp; &nbsp;Switch back to the Bloom application<br>
19. &nbsp; &nbsp;Click the Close button in the Bloom application.<br>
20. &nbsp; &nbsp;Verify that the Bloom application closes<br>
21. &nbsp; &nbsp;Double-click the Bloom icon on the Desktop<br>
22. &nbsp; &nbsp;Verify Bloom application starts<br>
23. &nbsp; &nbsp;Verify that the caption 'Feedback Disabled' appears in the About box while the Bloom application is opening</p>
<p>&nbsp;<br>
Cleanup Steps:<br>
1. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
