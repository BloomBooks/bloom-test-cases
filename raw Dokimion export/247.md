# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. Uninstall the Bloom application.<br>
2. Goto the windows Folder ".....\AppData\Local\BloomBetaInternal".<br>
3. Delete all the existing folders.</p>
<p>&nbsp;<br>
Action Steps:<br>
1. &nbsp; &nbsp;Download and Install the bloom application on the following location "https://bloomlibrary.org/installersBetaInternal/BloomInstaller.6.2.3087.BetaInternal.exe"<br>
2. &nbsp; &nbsp;Verify the Bloom application launched.<br>
3. &nbsp; &nbsp;Click the Settings button.<br>
4. &nbsp; &nbsp;Verify the Settings dialog appears.<br>
5. &nbsp; &nbsp;Click the Advanced Program Settings tab.<br>
6. &nbsp; &nbsp;Verify the checkbox &nbsp;"Automatically Update Bloom" is selected by default.<br>
7. &nbsp; &nbsp;Closed the Settings dialog.<br>
8. &nbsp; &nbsp;Click on the "Basic Book" Templates from the Sources For New Books.<br>
9. &nbsp; &nbsp;Verify the "MAKE A BOOK USING THIS SOURCE" button appears on the Right side pane.<br>
10. &nbsp; &nbsp;Click on the "MAKE A BOOK USING THIS SOURCE" button.<br>
11. &nbsp; &nbsp;Verify the Basic Book opens in the Edit Mode.<br>
12. &nbsp; &nbsp;Click on the Front Cover page from the Left pane.<br>
13. &nbsp; &nbsp;Verify the Front Cover page displays to Edit.<br>
14. &nbsp; &nbsp;Click on the Title Box from the Front Cover page<br>
15. &nbsp; &nbsp;Type Title as Test.<br>
16. &nbsp; &nbsp;Leave the Bloom application for 1 minute.<br>
17. &nbsp; &nbsp;Verify the Toast message "Downloading bloom updates" appears on the bottom right corner.<br>
18. &nbsp; &nbsp;Unplug the Internet connection.<br>
19. &nbsp; &nbsp;Verify the Bloom application does not crash.<br>
20. &nbsp; &nbsp;Close the Bloom application.<br>
21. &nbsp; &nbsp;Open the Bloom application.<br>
22. &nbsp; &nbsp;Click on the Get Help button from the Top Right corner.<br>
23. &nbsp; &nbsp;Verify the "Check For New Version" option appears.<br>
24. &nbsp; &nbsp;Click on the Check For New Version.<br>
25. &nbsp; &nbsp;Verify the following alert message appears: "Are you connected to the Internet?".<br>
26. &nbsp; &nbsp;Reconnect to the Internet connection.<br>
27. &nbsp; &nbsp;Click on the Get Help button from the Top Right corner.<br>
28. &nbsp; &nbsp;Verify the "Check For New Version" option appears.<br>
29. &nbsp; &nbsp;Click on the Check For New Version.<br>
30. &nbsp; &nbsp;Verify the Toast message "Downloading bloom updates" appears on the bottom right corner.<br>
31. &nbsp; &nbsp;Verify the Toast message "Applying updates" appears on the bottom right corner.<br>
32. &nbsp; &nbsp;wait until the Applying updates get completed.<br>
33. &nbsp; &nbsp;Verify the Toast message "Update for "...version number" is ready" appears on the bottom right corner.<br>
34. &nbsp; &nbsp;Verify the "Restart Bloom to Update" link appears on the Toast message.<br>
35. &nbsp; &nbsp;Click on the "Restart Bloom to Update" link.<br>
36. &nbsp; &nbsp;Verify the bloom gets restarted.<br>
37. &nbsp; &nbsp;Click on the Get Help button from the Top Right corner.<br>
38. &nbsp; &nbsp;Verify the "Check For New Version" option appears.<br>
39. &nbsp; &nbsp;Click on the Check For New Version<br>
40. &nbsp; &nbsp;Verify the Toast message "Your Bloom is up to date" appears on the bottom right corner.</p>
<p>&nbsp;<br>
Cleanup Steps:<br>
1. Delete the "Test" book.<br>
2. Close the Bloom application.</p>
<p>&nbsp;</p>
```
