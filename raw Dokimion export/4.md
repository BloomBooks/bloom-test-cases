# ID 4

# Name Troubleshooting page for errors

# Description

```html
est case for the Bloom application that displays the troubleshooting page when an error occurs
```

# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Right click on the BloomBetaInternal icon from the Desktop.<br>
2. &nbsp; &nbsp;Verify BloomBetaInternal Properties dialog opens.&nbsp;<br>
3. &nbsp; &nbsp;Copy the Target Location.<br>
4. &nbsp; &nbsp;Press Ctrl+E keys to open the File Explorer window.<br>
5. &nbsp; &nbsp;Paste the location on the Search bar. (C:\Users\UserName\AppData\Local\BloomBetaInternal\app-6.2.3120)<br>
6. &nbsp; &nbsp;Select the folder you just came from , it should have the format "app-[x].[y].[z]", where x.y.z is the Bloom version number.&nbsp;<br>
7. &nbsp; &nbsp;Right click that folder.&nbsp;<br>
8. &nbsp; &nbsp;Select Properties.<br>
9. &nbsp; &nbsp;Click the Security tab.<br>
10. &nbsp; &nbsp;Click Edit.<br>
11. &nbsp; &nbsp;Select the current user.<br>
12. &nbsp; &nbsp;Select &ldquo;Deny&rdquo; under Permissions for &lt;username&gt;<br>
13. &nbsp; &nbsp;Verify it should automatically check every other box under Deny as well.<br>
14. &nbsp; &nbsp;Click the Apply button.<br>
15. &nbsp; &nbsp;Verify Windows Security dialog opens.&nbsp;<br>
16. &nbsp; &nbsp;Click the Yes button.<br>
17. &nbsp; &nbsp;Verify Windows Security dialog closes.<br>
18. &nbsp; &nbsp;Click the OK button until the Properties window goes away.<br>
19. &nbsp; &nbsp;Close Bloom application.<br>
20. &nbsp; &nbsp;Download a different version of Bloom under the same channel. (eg., the most recent one).<br>
21. &nbsp; &nbsp;Double click it to install it.&nbsp;<br>
22. &nbsp; &nbsp;Verify an error message appears saying that 'there was an installation error".&nbsp;<br>
23. &nbsp; &nbsp;Verify one of the buttons on it should link to a page on the community site titled "How to fix installation problems."</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Right click that folder. (eg., app-6.2.3120 from C:\Users\UserName\AppData\Local\BloomBetaInternal)<br>
2. &nbsp; &nbsp;Select Properties.<br>
3. &nbsp; &nbsp;Click the Security tab.<br>
4. &nbsp; &nbsp;Click Edit.<br>
5. &nbsp; &nbsp;Select the current user.<br>
6. &nbsp; &nbsp;Select &ldquo;Remove&rdquo; under Permissions for &lt;username&gt;<br>
7. &nbsp; &nbsp;Verify it should automatically uncheck every other box under Deny as well.<br>
8. &nbsp; &nbsp;Click the Apply button.<br>
9. &nbsp; &nbsp;Click the Yes button.<br>
10. &nbsp; &nbsp;Click the OK button.&nbsp;<br>
11. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
