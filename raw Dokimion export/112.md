# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Type Language settings in the Windows Search bar.&nbsp;<br>
2. &nbsp; &nbsp;Verify Language settings option appears.<br>
3. &nbsp; &nbsp;Click Language settings option<br>
4. &nbsp; &nbsp;Verify Language settings window opens.&nbsp;<br>
5. &nbsp; &nbsp;Click the 'Add a language' button under Preferred languages.&nbsp;<br>
6. &nbsp; &nbsp;Verify Choose a language to install dialog opens.&nbsp;<br>
7. &nbsp; &nbsp;Enter "spanish mexico" in the Search bar.&nbsp;<br>
8. &nbsp; &nbsp;Verify Spanish Mexico language appears.<br>
9. &nbsp; &nbsp;Select Spanish Mexico language<br>
10. &nbsp; &nbsp;Click the Next button.&nbsp;<br>
11. &nbsp; &nbsp;Click the Install button.&nbsp;<br>
12. &nbsp; &nbsp;Verify the installation completed.&nbsp;<br>
13. &nbsp; &nbsp;Restart Windows.<br>
14. &nbsp; &nbsp;Verify &lsquo;ESP&rsquo; Spanish Mexico language appears as the windows system language.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Download the Spanish book from the given link: https://bloomlibrary.org/book/Nx80gRwuiB&nbsp;<br>
2. &nbsp; &nbsp;Verify the book opens in the Bloom application.<br>
3. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
4. &nbsp; &nbsp;Verify the book opens in the Edit mode.&nbsp;<br>
5. &nbsp; &nbsp;Click the Credits page.&nbsp;<br>
6. &nbsp; &nbsp;Verify the Copyright Information should appear in Spanish language.<br>
7. &nbsp; &nbsp;Click 'Basic Book' under Templates. (eg., Libro basico under Plantillas)<br>
8. &nbsp; &nbsp;Click CREAR UN LIBRO UTILIZANDO ESTE LIBRO FUENTE button.<br>
9. &nbsp; &nbsp;Verify the book opens in the edit mode.&nbsp;<br>
10. &nbsp; &nbsp;Click the Credits page (ie. Pagina de creditos page thumbnail).<br>
11. &nbsp; &nbsp;Click the 'Needs Copyright' button.<br>
12. &nbsp; &nbsp;Verify Copyright and License dialog opens.&nbsp;<br>
13. &nbsp; &nbsp;Enter 'Tester' in the Copyright Holder text box.&nbsp;<br>
14. &nbsp; &nbsp;Click the OK button.&nbsp;<br>
15. &nbsp; &nbsp;Verify the Copyright information appears in Spanish.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Uninstall Spanish Mexico language.<br>
2. &nbsp; &nbsp;Delete the Spanish book.<br>
3. &nbsp; &nbsp;Delete the Basic book.<br>
4. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
