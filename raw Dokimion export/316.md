# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click the 'Leveled Reader' book under Specialized Templates heading. (which appears at the lower left pane in Collectins view)<br>
2. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
3. &nbsp; &nbsp;Verify the book opens in the Edit mode.&nbsp;<br>
4. &nbsp; &nbsp;Click the Add Page button.<br>
5. &nbsp; &nbsp;Verify the Add Page dialog opens.&nbsp;<br>
6. &nbsp; &nbsp;Click the 'Just text' template from Basic Pages.&nbsp;<br>
7. &nbsp; &nbsp;Click the Add Page button.<br>
8. &nbsp; &nbsp;Verify 'Just text' page is added.&nbsp;<br>
9. &nbsp; &nbsp;Click on the text box.<br>
10. &nbsp; &nbsp;Type 'There' in the text box.&nbsp;<br>
11. &nbsp; &nbsp;Verify the Word count appears '1' on the Leveled Reader Tool menu.&nbsp;<br>
12. &nbsp; &nbsp;Type 'Once' in the text box.<br>
13. &nbsp; &nbsp;Verify the Word count appears '2' on the Leveled Reader Tool menu.&nbsp;<br>
14. &nbsp; &nbsp;Type 'was a boy' in the text box.<br>
15. &nbsp; &nbsp;Verify the Word count appears '5' on the Leveled Reader Tool menu.&nbsp;<br>
16. &nbsp; &nbsp;Click the triangle button near the Page 1 thumbnail.<br>
17. &nbsp; &nbsp;Verify a popup menu opens.&nbsp;<br>
18. &nbsp; &nbsp;Click the 'Remove Page' option.<br>
19. &nbsp; &nbsp;Verify a warning message 'Really Remove Page?' appears in the dialog.<br>
20. &nbsp; &nbsp;Click the 'Remove' button.<br>
21. &nbsp; &nbsp;Verify the Word count Status is changed to '0'.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the Leveled Reader book.<br>
2. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
