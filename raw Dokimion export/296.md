&nbsp;<br>
Action Steps:</p>
<p>1. &nbsp; &nbsp;Click on the "Basic Book" Template from the Sources For New Book pane.<br>
2. &nbsp; &nbsp;Verify that the "Basic Book" preview appears on the Right side pane.<br>
3. &nbsp; &nbsp;Click on the "MAKE A BOOK USING THIS SOURCE" button.<br>
4. &nbsp; &nbsp;Verify that the &ldquo;Basic Book&rdquo; opens in the Edit tab<br>
5. &nbsp; &nbsp;Click the &lsquo;Add Page&rsquo; button at the bottom left corner of the page<br>
6. &nbsp; &nbsp;Verify that the &ldquo;Add Page..&rdquo; dialog box is displayed.<br>
7. &nbsp; &nbsp;Select the "Basic text &amp; picture" Template under the &ldquo;Basic Pages&rdquo; option.<br>
8. &nbsp; &nbsp;Click the &ldquo;ADD PAGE&rdquo; button.<br>
9. &nbsp; &nbsp;Verify that the &ldquo;Add Page..&rdquo; dialog box is closed.<br>
10. &nbsp; &nbsp;Verify "Basic text &amp; picture" layout is displayed on the right side<br>
11. &nbsp; &nbsp;Hover the mouse over the picture icon in the "Front cover" page.<br>
12. &nbsp; &nbsp;Verify that the "Change image" button is displayed.<br>
13. &nbsp; &nbsp;Click the "Change image' button in the picture pane.<br>
14. &nbsp; &nbsp;Verify the "Image Toolbox" dialog box opens.<br>
15. &nbsp; &nbsp;Click the File button.<br>
16. &nbsp; &nbsp;Verify "Choose Picture Files" dialog box opens.&nbsp;<br>
17. &nbsp; &nbsp;Select 'Motor.png' from the picture folder<br>
18. &nbsp; &nbsp;Click the Open Key.<br>
19. &nbsp; &nbsp;Verify that the selected image appears on the "Image Toolbox" dialog box.<br>
20. &nbsp; &nbsp;Click on the OK button<br>
21. &nbsp; &nbsp;Verify that the "Image Toolbox" dialog box is closed<br>
22. &nbsp; &nbsp;Verify the selected image appears on the "Basic text &amp; picture" page<br>
23. &nbsp; &nbsp;Click on the added image<br>
24. &nbsp; &nbsp;Verify that crop handle appears on all four sides of the image<br>
25. &nbsp; &nbsp;Click on the Top Crop handle&nbsp;<br>
26. &nbsp; &nbsp;Drag the crop handle downward<br>
27. &nbsp; &nbsp;Verify that the picture is cropped correctly from the top without any errors in the Bloom application.<br>
28. &nbsp; &nbsp;Click on the Bottom Crop handle&nbsp;<br>
29. &nbsp; &nbsp;Drag the crop handle upward<br>
30. &nbsp; &nbsp;Verify that the picture is cropped correctly from the bottom without any errors in the Bloom application.<br>
31. &nbsp; &nbsp;Click on the Left Crop handle.<br>
32. &nbsp; &nbsp;Drag the crop handle toward the right.<br>
33. &nbsp; &nbsp;Verify that the picture is cropped correctly from the left without any errors in the Bloom application.<br>
34. &nbsp; &nbsp;Click on the Right Crop handle.<br>
35. &nbsp; &nbsp;Drag the crop handle toward the left.<br>
36. &nbsp; &nbsp;Verify that the picture is cropped correctly from the right without any errors in the Bloom application.</p>
<p><br>
Cleanup Steps:<br>
1. &nbsp;Delete the downloaded &ldquo;Crop Error.png&rdquo; image.<br>
2. Delete the &ldquo;Basic Book&rdquo;<br>
3. &nbsp;Close Bloom application.</p>
