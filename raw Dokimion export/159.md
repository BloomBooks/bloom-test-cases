<p>Setup Steps:<br>
1. &nbsp; &nbsp;Connect Android Mobile to the Computer via USB Cable.<br>
2. &nbsp; &nbsp;Verify the Bloom Reader installed in the Android Mobile.<br>
3. &nbsp; &nbsp;Download MC - YRT- Yumi Read Level 3 -ETP from https://drive.google.com/drive/folders/1z0f7PdQywE8azia4qeEvRsU5v3doHM4O (Needs Developer's approval) in the Downloads folder.<br>
4. &nbsp; &nbsp;Right click on the .zip folder.&nbsp;<br>
5. &nbsp; &nbsp;Verify a popup menu appears.<br>
6. &nbsp; &nbsp;Click the 'Extract Here' option.<br>
7. &nbsp; &nbsp;Verify the extracted MC - YRT- Yumi Read Level 3 -ETP folder opens.&nbsp;<br>
8. &nbsp; &nbsp;Double click on MC - YRT - Yumi Read Level 3 - ETP folder.&nbsp;<br>
9. &nbsp; &nbsp;Verify the folder opens.&nbsp;<br>
10. &nbsp; &nbsp;Double click on MC -YRT - Yumi Read Level 3 - ETP.Bloom Book Collection.<br>
11. &nbsp; &nbsp;Verify the book opens in the Bloom application.<br>
12. &nbsp; &nbsp;Close the Bloom application.<br>
13. &nbsp; &nbsp;Launch Bloom application<br>
14. &nbsp; &nbsp;Click the Settings button.<br>
15. &nbsp; &nbsp;Verify the Settings dialog opens.&nbsp;<br>
16. &nbsp; &nbsp;Click the Bloom Subscription tab.<br>
17. &nbsp; &nbsp;Verify the Bloom Subscription tab opens.&nbsp;<br>
18. &nbsp; &nbsp;Enter 'Test-727011-1339' in the Subscription code.&nbsp;<br>
19. &nbsp; &nbsp;Verify the Restart button appears instead of the OK button.<br>
20. &nbsp; &nbsp;Click the Restart button.<br>
21. &nbsp; &nbsp;Verify Bloom gets restarted.<br>
22. &nbsp; &nbsp;Verify the MC -YRT - Yumi Read Level 3 - ETP book collection opens in the Bloom application.&nbsp;<br>
23. &nbsp; &nbsp;Verify there are 22 books appearing under the book collection.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click the Publish tab.<br>
2. &nbsp; &nbsp;Click the BloomPUB button.<br>
3. &nbsp; &nbsp;Verify Publish as BloomPUB view opens.&nbsp;<br>
4. &nbsp; &nbsp;Click 'Make All BloomPUBs from Collection' button (which appears at the bottom right corner of the page)<br>
5. &nbsp; &nbsp;Verify Make All BloomPUBs from Collection dialog opens.&nbsp;<br>
6. &nbsp; &nbsp;Verify 'Compress into single.bloombundle file' checkbox is selected.<br>
7. &nbsp; &nbsp;Click the 'Make' button.<br>
8. &nbsp; &nbsp;Verify Bulk Save BloomPUBs dialog opens.&nbsp;<br>
9. &nbsp; &nbsp;Wait for a few minutes.<br>
10. &nbsp; &nbsp;Verify BloomPubs window opens.&nbsp;<br>
11. &nbsp; &nbsp;Verify the BloomPubs file saved location. (eg.,C:\Users\UserName)\AppData\Local\Temp\BloomPubs)<br>
12. &nbsp; &nbsp;Click the Close button.<br>
13. &nbsp; &nbsp;Verify the BloomPubs window closes.&nbsp;<br>
14. &nbsp; &nbsp;Click the Close button.<br>
15. &nbsp; &nbsp;Verify the Bulk Save BloomPubs dialog closes.&nbsp;<br>
16. &nbsp; &nbsp;Copy the 'BloomPubs' folder from the given location (ie., C:\Users\UserName)\AppData\Local\Temp\BloomPubs)<br>
17. &nbsp; &nbsp;Open Android Mobile Internal Storage. (eg., This PC\AndroidMobile\Internal shared storage) in the computer.<br>
18. &nbsp; &nbsp;Double click on the Bloombundle folder.&nbsp;<br>
19. &nbsp; &nbsp;Paste the 'BloomPubs' folder.&nbsp;<br>
20. &nbsp; &nbsp;Click the close button.<br>
21. &nbsp; &nbsp;Verify the Bloombundle window closes.<br>
22. &nbsp; &nbsp;Open Bloom Reader in the Android Mobile.&nbsp;<br>
23. &nbsp; &nbsp;Click the Hamburger button (which appears at the top left corner of the page)<br>
24. &nbsp; &nbsp;Verify an option menu opens.<br>
25. &nbsp; &nbsp;Click the 'Open BloomBundle File' option.<br>
26. &nbsp; &nbsp;Click the File manager of the Android Mobile.<br>
27. &nbsp; &nbsp;Verify the BloomPubs folder appears.<br>
28. &nbsp; &nbsp;Tap on the 'BloomPubs' folder.<br>
29. &nbsp; &nbsp;Verify the .bloombundle file appears.<br>
30. &nbsp; &nbsp;Tap on .bloombundle file<br>
31. &nbsp; &nbsp;Verify all the 22 books from &lsquo;MC -YRT - Yumi Read Level 3 - ETP&rsquo; book collection should appear on the Bloom page.&nbsp;</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Remove the USB cable &nbsp;from the Computer.<br>
2. &nbsp; &nbsp;Delete the Subscription Code.<br>
3. &nbsp; &nbsp;Delete &lsquo;MC -YRT - Yumi Read Level 3 - ETP&rsquo; book.<br>
4. &nbsp; &nbsp;Close Bloom Reader application.<br>
5. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
