### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Download and Install Keyman Keyboard for Windows from the given link : https://downloads.keyman.com/windows/stable/18.0.243/keyman-18.0.243.exe<br>
2. &nbsp; &nbsp;Download and Install UTB Unicode.kmx keyboard from the given link : https://drive.google.com/drive/u/0/folders/1a6McYeQA5MKyFvwlUcsxaAM-MfjCIj51<br>
3. &nbsp; &nbsp;Launch Bloom application.&nbsp;</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click the Basic book under Templates.<br>
2. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
3. &nbsp; &nbsp;Verify the book opens in the edit mode.<br>
4. &nbsp; &nbsp;Click the Add Page button.<br>
5. &nbsp; &nbsp;Verify Add page dialog opens.<br>
6. &nbsp; &nbsp;Click 'Just a text' template.<br>
7. &nbsp; &nbsp;Click the Add page button.<br>
8. &nbsp; &nbsp;Verify the Just a text template is added.<br>
9. &nbsp; &nbsp;Click the Keyboard icon from the taskbar.<br>
10. &nbsp; &nbsp;Click the Keyman keyboard.<br>
11. &nbsp; &nbsp;Switch to 'UTB Unicode' keyboard.<br>
12. &nbsp; &nbsp;Long press 'Space bar' key. (or any character keys)<br>
13. &nbsp; &nbsp;Verify the long press feature is disabled.<br>
14. &nbsp; &nbsp;Type 'a in the text box.<br>
15. &nbsp; &nbsp;Verify á should appear on the text box.<br>
16. &nbsp; &nbsp;Type x'a<br>
17. &nbsp; &nbsp;Verify áxá &nbsp;should appear on the text box.<br>
18. &nbsp; &nbsp;Switch to English Keyman keyboard.<br>
19. &nbsp; &nbsp;Long press 'Space bar' key. (or any character keys)<br>
20. &nbsp; &nbsp;Verify the long press feature is enabled.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Uninstall UTB Unicode Keyboard.&nbsp;<br>
2. &nbsp; &nbsp;Uninstall Keyman Keyboard.&nbsp;<br>
3. &nbsp; &nbsp;Delete the Basic book.<br>
4. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
