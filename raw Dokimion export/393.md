# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click a Basic book under Templates.<br>
2. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
3. &nbsp; &nbsp;Verify the book opens in the edit view.&nbsp;<br>
4. &nbsp; &nbsp;Enter 'Test Book' in the Front cover title box.&nbsp;<br>
5. &nbsp; &nbsp;Click the Collections tab.<br>
6. &nbsp; &nbsp;Verify the Collections view appears.<br>
7. &nbsp; &nbsp;Right click on the 'Test Book' thumbnail under Book Collections.<br>
8. &nbsp; &nbsp;Verify a context menu opens.<br>
9. &nbsp; &nbsp;Click 'Show in File Explorer'<br>
10. &nbsp; &nbsp;Verify the book appears in the File Explorer window. (C:\Users\UserName\Documents\Bloom\Book Collections)<br>
11. &nbsp; &nbsp;Double Click on the Book's folder.&nbsp;<br>
12. &nbsp; &nbsp;Verify the book folder opens.<br>
13. &nbsp; &nbsp;Verify a 'placeHolder' file appears in the folder.&nbsp;<br>
14. &nbsp; &nbsp;Minimize the explorer window.<br>
15. &nbsp; &nbsp;Open the Downloads folder. (eg., C:\Users\UserName\Downloads)<br>
16. &nbsp; &nbsp;Open '01_03_16x9.png' picture file in the Paint application.<br>
17. &nbsp; &nbsp;Press Ctrl+A to select the image.<br>
18. &nbsp; &nbsp;Press Ctrl+C to copy the image.<br>
19. &nbsp; &nbsp;Click the Front Cover page in the Bloom.<br>
20. &nbsp; &nbsp;Click the ellipse (...) button.&nbsp;<br>
21. &nbsp; &nbsp;Verify the option menu opens.&nbsp;<br>
22. &nbsp; &nbsp;Click the 'Paste Image' option.<br>
23. &nbsp; &nbsp;Verify the '01_03_16x9.png'picture image appears on the Front cover picture pane.<br>
24. &nbsp; &nbsp;Click the ellipse (...) button.&nbsp;<br>
25. &nbsp; &nbsp;Verify the option menu opens.&nbsp;<br>
26. &nbsp; &nbsp;Click the 'Copy Image' option.<br>
27. &nbsp; &nbsp;Open a Paint application.&nbsp;<br>
28. &nbsp; &nbsp;Press Ctrl+V keys.<br>
29. &nbsp; &nbsp;Verify the sample picture appears on the Paint application.&nbsp;<br>
30. &nbsp; &nbsp;Click the ellipse (...) button in the Front cover image pane. (in bloom).&nbsp;<br>
31. &nbsp; &nbsp;Verify the option menu opens.&nbsp;<br>
32. &nbsp; &nbsp;Click the 'Delete' option.<br>
33. &nbsp; &nbsp;Verify the image is deleted from the picture pane.&nbsp;<br>
34. &nbsp; &nbsp;Open the Downloads folder. (eg., C:\Users\UserName\Downloads)<br>
35. &nbsp; &nbsp;Open '02_10_16x9.png' file in the Paint application.<br>
36. &nbsp; &nbsp;Press Ctrl+A to select the image.<br>
37. &nbsp; &nbsp;Press Ctrl+C to copy the image.<br>
38. &nbsp; &nbsp;Click on the Front cover picture pane. (in Bloom)<br>
39. &nbsp; &nbsp;Click on the 'Paste image' button.<br>
40. &nbsp; &nbsp;Verify the sample picture appears on the Front Cover picture pane.<br>
41. &nbsp; &nbsp;Maximize the File Explorer window.<br>
42. &nbsp; &nbsp;Verify that only one placeholder file is shown in the folder.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the Sample pictures from the Downloads folder.&nbsp;<br>
2. &nbsp; &nbsp;Delete the Basic book.<br>
3. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
