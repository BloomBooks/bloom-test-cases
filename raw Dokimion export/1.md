<p>Setup Steps:<br>

2. &nbsp; &nbsp;Type Language settings in the Windows Search bar.&nbsp;<br>
3. &nbsp; &nbsp;Verify Language settings option appears.<br>
4. &nbsp; &nbsp;Click Language settings option<br>
5. &nbsp; &nbsp;Verify Language settings window opens.&nbsp;<br>
6. &nbsp; &nbsp;Click the 'Add a language' button under Preferred languages.&nbsp;<br>

8. &nbsp; &nbsp;Enter "spanish mexico" in the Search bar.&nbsp;<br>
9. &nbsp; &nbsp;Verify Spanish Mexico language appears.<br>
10. &nbsp; &nbsp;Select Spanish Mexico language<br>
11. &nbsp; &nbsp;Click the Next button.&nbsp;<br>
12. &nbsp; &nbsp;Click the Install button.&nbsp;<br>
13. &nbsp; &nbsp;Verify the installation completed.&nbsp;<br>
14. &nbsp; &nbsp;Restart Windows.<br>
15. &nbsp; &nbsp;Verify &lsquo;ESP&rsquo; Spanish Mexico language appears as the windows system language.&nbsp;</p>
<p>&nbsp;</p>

1. &nbsp; &nbsp;Double click on the Bloom Alpha build.<br>
2. &nbsp; &nbsp;Verify "Bloom Alpha has missing system dependencies' dialog.&nbsp;<br>
3. &nbsp; &nbsp;Click the Install button.<br>
4. &nbsp; &nbsp;Verify 'Installing ..' dialog appears.&nbsp;<br>
5. &nbsp; &nbsp;Verify Bloom Alpha dialog appears with 'I accept terms of the..' button.&nbsp;<br>
6. &nbsp; &nbsp;Verify Bloom Alpha installed successfully.<br>
7. &nbsp; &nbsp;Verify Bloom Alpha (open/create collections) dialog appears with Spanish UI language.</p>

<p>Cleanup Steps:<br>
