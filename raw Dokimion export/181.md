<p>Setup:<br>

Action Steps:</p>
<p>1. &nbsp; &nbsp;Click on the "Basic Book" Template from the Sources For New Book pane.<br>
2. &nbsp; &nbsp;Verify that the "Basic Book" preview appears on the Right side pane.<br>
3. &nbsp; &nbsp;Click on the "MAKE A BOOK USING THIS SOURCE" button.<br>
4. &nbsp; &nbsp;Verify that the &ldquo;Basic Book&rdquo; Edit Pane appears.<br>
5. &nbsp; &nbsp;Click the &lsquo;Add Page&rsquo; button at the bottom left of the page.<br>
6. &nbsp; &nbsp;Verify that the &ldquo;Add Page..&rdquo; dialog box is displayed.<br>
7. &nbsp; &nbsp;Select the &ldquo;Picture &amp; Word&rdquo; page under the &ldquo;Basic Pages&rdquo; option.<br>
8. &nbsp; &nbsp;Click the &ldquo;ADD PAGE&rdquo; button.<br>
9. &nbsp; &nbsp;Verify that the &ldquo;Add Page..&rdquo; dialog box is closed.<br>
10. &nbsp; &nbsp;Verify that the new page is displayed.<br>
11. &nbsp; &nbsp;Click the Hamburger menu from the top right corner.<br>
12. &nbsp; &nbsp;Verify the Talking Book Tool appears.<br>
13. &nbsp; &nbsp;Click on the Text box from Page 1.<br>
14. &nbsp; &nbsp;Type "Test Talking Book Tool".<br>
