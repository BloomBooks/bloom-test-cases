## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Open Notepad application.<br>
2. &nbsp; &nbsp;Enter &nbsp;'This is the first sentence'<br>
3. &nbsp; &nbsp;Press the Enter Key.<br>
4. &nbsp; &nbsp;Press the Enter key. (to create a page break)<br>
5. &nbsp; &nbsp;Enter 'This is the second sentence'<br>
6. &nbsp; &nbsp;Press the Enter Key.<br>
7. &nbsp; &nbsp;Press the Enter key.(to create a page break)<br>
8. &nbsp; &nbsp;Enter 'This is the third sentence'<br>
9. &nbsp; &nbsp;Press the Enter Key.<br>
10. &nbsp; &nbsp;Press the Enter key.(to create a page break)<br>
11. &nbsp; &nbsp;Copy all the Sentences with paragraph breaks from Notepad.<br>
12. &nbsp; &nbsp;Launch Bloom application.<br>
13. &nbsp; &nbsp;Click a Basic Book.<br>
14. &nbsp; &nbsp;Verify MAKE A BOOK USING THIS SOURCE button available on the right pane.&nbsp;<br>
15. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
16. &nbsp; &nbsp;Verify the Edit view opens.<br>
17. &nbsp; &nbsp;Click the Add page button.<br>
18. &nbsp; &nbsp;Verify the Add page dialog opens.&nbsp;<br>
19. &nbsp; &nbsp;Select 'Just a text' template.&nbsp;<br>
20. &nbsp; &nbsp;Click the Add page button.<br>
21. &nbsp; &nbsp;Verify Just a text page is added.&nbsp;<br>
22. &nbsp; &nbsp;Paste the Sentence on the text page.&nbsp;<br>
23. &nbsp; &nbsp;Click the Credit Page thumbnail.<br>
24. &nbsp; &nbsp;Verify the Credits page opens.&nbsp;<br>
25. &nbsp; &nbsp;Switch back to Page 1 thumbnail.<br>
26. &nbsp; &nbsp;Verify the paragraph breaks should be retained. (ie., it should appear like in the Notepad)<br>
27. &nbsp; &nbsp;Open Word application.<br>
28. &nbsp; &nbsp;Enter &nbsp;'This is the first sentence'<br>
29. &nbsp; &nbsp;Press the Enter Key.<br>
30. &nbsp; &nbsp;Press the Enter key. (to create a page break)<br>
31. &nbsp; &nbsp;Enter 'This is the second sentence'<br>
32. &nbsp; &nbsp;Press the Enter Key.<br>
33. &nbsp; &nbsp;Press the Enter key.(to create a page break)<br>
34. &nbsp; &nbsp;Enter 'This is the third sentence'<br>
35. &nbsp; &nbsp;Press the Enter Key.<br>
36. &nbsp; &nbsp;Press the Enter key.(to create a page break)<br>
37. &nbsp; &nbsp;Copy all the Sentences with paragraph breaks from Word.<br>
38. &nbsp; &nbsp;Open Bloom application.<br>
39. &nbsp; &nbsp;Click the Basic Book.<br>
40. &nbsp; &nbsp;Verify EDIT THIS BOOK button available on the right pane.&nbsp;<br>
41. &nbsp; &nbsp;Click the EDIT THIS BOOK button.<br>
42. &nbsp; &nbsp;Verify the edit view opens.<br>
43. &nbsp; &nbsp;Click the Add page button.<br>
44. &nbsp; &nbsp;Verify the Add page dialog opens.&nbsp;<br>
45. &nbsp; &nbsp;Select 'Just a text' template.&nbsp;<br>
46. &nbsp; &nbsp;Click the Add page button.<br>
47. &nbsp; &nbsp;Verify Just a text page is added.&nbsp;<br>
48. &nbsp; &nbsp;Paste the Sentence on the text page.&nbsp;<br>
49. &nbsp; &nbsp;Click the Credit Page thumbnail.<br>
50. &nbsp; &nbsp;Verify the Credits page opens.&nbsp;<br>
51. &nbsp; &nbsp;Switch back to Page 1 thumbnail.<br>
52. &nbsp; &nbsp;Verify the paragraph breaks should be retained. (ie., it should appear like in the Word application)</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
```
