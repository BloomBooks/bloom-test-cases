# ID 541

# Name Failing to load a .bloomCollection results in a report including all files

# Description

```html
<p>Failing to load a .bloomCollection then restarting Bloom results in a report including all files in that directory (BL-13276)</p>
```

# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom Alpha build.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Close the Alpha build. (This makes it so the Alpha will be the channel which is run when opening a .bloomCollection file)<br>
2. &nbsp; &nbsp;Press Ctrl+E keys.<br>
3. &nbsp; &nbsp;Verify the File Explorer window opens.&nbsp;<br>
4. &nbsp; &nbsp;Click the Documents option from the left pane.&nbsp;<br>
5. &nbsp; &nbsp;Verify the Documents window opens.<br>
6. &nbsp; &nbsp;Double click on the Bloom folder.&nbsp;<br>
7. &nbsp; &nbsp;Verify the Bloom folder opens.&nbsp;<br>
8. &nbsp; &nbsp;Right click on any one of the Bloom Book collection file names. (eg., North Ambrym Super Bible.BloomBookCollection)<br>
9. &nbsp; &nbsp;Verify an option menu opens.<br>
10. &nbsp; &nbsp;Click the 'Open With..' option.&nbsp;<br>
11. &nbsp; &nbsp;Verify a submenu opens.&nbsp;<br>
12. &nbsp; &nbsp;Click the 'Notepad' option.<br>
13. &nbsp; &nbsp;Verify the file opens in the Notepad application.<br>
14. &nbsp; &nbsp;Press Ctrl+A to select all.<br>
15. &nbsp; &nbsp;Press the Delete key to delete all the contents.&nbsp;<br>
16. &nbsp; &nbsp;Click the File menu.<br>
17. &nbsp; &nbsp;Verify the file menu opens.&nbsp;<br>
18. &nbsp; &nbsp;Click the Save option.<br>
19. &nbsp; &nbsp;Verify the (corrupted) file is saved.<br>
20. &nbsp; &nbsp;Verify the (corrupted) file size is changed to 1 KB. (or less)<br>
21. &nbsp; &nbsp;Double click on the corrupted .Bloom Book collection file name&nbsp;<br>
22. &nbsp; &nbsp;Verify BloomAlpha Problem dialog opens.&nbsp;<br>
23. &nbsp; &nbsp;Click the Report button.<br>
24. &nbsp; &nbsp;Verify another message 'Successfully reported issue:https://issues.bloomlibrary.org/youtrack/issue/BL-xxxxx)appears in the dialog with OK button.<br>
25. &nbsp; &nbsp;Click the OK button.<br>
26. &nbsp; &nbsp;Enter "https://issues.bloomlibrary.org/youtrack/issues" in the browser window.<br>
27. &nbsp; &nbsp;Verify YouTrack opens.&nbsp;<br>
28. &nbsp; &nbsp;Click the Issues option from the left pane.&nbsp;<br>
29. &nbsp; &nbsp;Verify the newly created issue appears on the right pane.&nbsp;<br>
30. &nbsp; &nbsp;Click on the Error Report link.&nbsp;<br>
31. &nbsp; &nbsp;Verify the corrupt .bloomCollection file should appear in the attachments section.<br>
32. &nbsp; &nbsp;Verify no other files from the same folder should show up in the attachments section.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the .BloomBook Collection.<br>
2. &nbsp; &nbsp;Close Youtrack application.<br>
3. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
