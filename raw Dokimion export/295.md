# Steps

## Step

### Action

```html
<p>Setup Steps:</p>
<p>1. Create a new blank Word document.<br>
2. Insert images in PNG or JPEG format.<br>
3. Save the document as "test.docx".<br>
4. Close and reopen the previously saved Word document to confirm it was saved properly.<br>
5. Launch the Bloom application.<br>
&nbsp;<br>
Action Steps:</p>
<p>1. &nbsp; &nbsp;Click on the "Basic Book" Template from the Sources For New Book pane.<br>
2. &nbsp; &nbsp;Verify that the "Basic Book" preview appears on the Right side pane.<br>
3. &nbsp; &nbsp;Click on the "MAKE A BOOK USING THIS SOURCE" button.<br>
4. &nbsp; &nbsp;Verify that the &ldquo;Basic Book&rdquo; Edit Pane appears.<br>
5. &nbsp; &nbsp;Click the Add Page button.<br>
6. &nbsp; &nbsp;Verify the Add Page dialog opens.<br>
7. &nbsp; &nbsp;Click the Basic text &amp; picture template.<br>
8. &nbsp; &nbsp;Click the Add Page button.<br>
9. &nbsp; &nbsp;Verify that the &ldquo;Add Page..&rdquo; dialog box is closed.<br>
10. &nbsp; &nbsp;Verify that the new &ldquo;Custom&rdquo; page is displayed.<br>
11. &nbsp; &nbsp;Hover the mouse over the picture icon on page 1.<br>
12. &nbsp; &nbsp;Verify that the "Change image" button is displayed.<br>
13. &nbsp; &nbsp;Click the "Change image' button in the picture pane.<br>
14. &nbsp; &nbsp;Verify the "Image Toolbox" dialog box opens.<br>
15. &nbsp; &nbsp;Click the File button.<br>
16. &nbsp; &nbsp;Verify "Choose Picture Files" dialog box opens.&nbsp;<br>
17. &nbsp; &nbsp;Click the valid ".png" image from the "Choose Picture Files" dialog box.<br>
18. &nbsp; &nbsp;Click the Open Key.<br>
19. &nbsp; &nbsp;Verify that the selected image appears on the "Image Toolbox" dialog box.<br>
20. &nbsp; &nbsp;Click the OK button.<br>
21. &nbsp; &nbsp;Verify that the selected picture appears on page 1.<br>
22. &nbsp; &nbsp;Click the Add Page button.<br>
23. &nbsp; &nbsp;Verify the Add Page dialog opens.<br>
24. &nbsp; &nbsp;Click the Basic text &amp; picture template.<br>
25. &nbsp; &nbsp;Click the Add Page button.<br>
26. &nbsp; &nbsp;Verify that the &ldquo;Add Page..&rdquo; dialog box is closed.<br>
27. &nbsp; &nbsp;Verify that the new &ldquo;Custom&rdquo; page is displayed.<br>
28. &nbsp; &nbsp;Select Page 1.<br>
29. &nbsp; &nbsp;right click on the inserted image.<br>
30. &nbsp; &nbsp;Click the &ldquo;Copy image&rdquo; from the submenu.<br>
31. &nbsp; &nbsp;Select Page 2.<br>
32. &nbsp; &nbsp;right click on the pictures area.<br>
33. &nbsp; &nbsp;Click the &ldquo;Paste &nbsp;image&rdquo; from the submenu.<br>
34. &nbsp; &nbsp;Verify that the copied image is pasted in page 2.<br>
35. &nbsp; &nbsp;Open the test.docx document.<br>
36. &nbsp; &nbsp;Cut the image from the document.<br>
37. &nbsp; &nbsp;In the Bloom application, select Page 1 in the left-side Pages pane.<br>
38. &nbsp; &nbsp;right click on the image.<br>
39. &nbsp; &nbsp;Click the &ldquo;Paste &nbsp;image&rdquo; from the submenu.<br>
40. &nbsp; &nbsp;Verify that the image cut from the word document is pasted in page 1.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. Delete the Basic Book.<br>
2. Close Bloom application.</p>
<p>&nbsp;</p>
```
