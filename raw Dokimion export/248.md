# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Press Windows + I keys.<br>
2. &nbsp; &nbsp;Click the Time &amp; Language button.&nbsp;<br>
3. &nbsp; &nbsp;Verify the Settings dialog opens.&nbsp;<br>
4. &nbsp; &nbsp;Click Language option (from the left pane)<br>
5. &nbsp; &nbsp;Click Add a language (+) button.&nbsp;<br>
6. &nbsp; &nbsp;Verify Choose a language to install dialog opens.&nbsp;<br>
7. &nbsp; &nbsp;Enter 'Netherlands' in the search bar.&nbsp;<br>
8. &nbsp; &nbsp;Select 'Dutch (Netherlands)' language.&nbsp;<br>
9. &nbsp; &nbsp;Click the Next button.<br>
10. &nbsp; &nbsp;Select 'Set as my Windows display language' under Install language pack .<br>
11. &nbsp; &nbsp;Click the Install button.&nbsp;<br>
12. &nbsp; &nbsp;Verify the Dutch language is installed.&nbsp;<br>
13. &nbsp; &nbsp;Verify a warning message 'Setting Windows display language' appears in the dialog.&nbsp;<br>
14. &nbsp; &nbsp;Click the 'No, Sign out later' button.&nbsp;<br>
15. &nbsp; &nbsp;Restart Windows OS.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Download the older version of BloomInstaller.exe (eg., BloomInstaler.5.6.516) from blorg.&nbsp;<br>
2. &nbsp; &nbsp;Double click on the BloomInstaller.exe file.&nbsp;<br>
3. &nbsp; &nbsp;Verify 'Choose a Language' dialog appears.<br>
4. &nbsp; &nbsp;Verify a warning message 'Our apologies, this program has not yet been localized for Dutch (Nederlands). Please, choose from one of the following languages..' appears in the dialog.&nbsp;<br>
5. &nbsp; &nbsp;Click the OK button.<br>
6. &nbsp; &nbsp;Verify that the Bloom was successfully installed in the system.&nbsp;</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Remove the Dutch (Netherlands) from system language.&nbsp;<br>
2. &nbsp; &nbsp;Close Bloom application.&nbsp;</p>
<p>&nbsp;</p>
```
