## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Open LibreOffice Writer document.&nbsp;<br>
2. &nbsp; &nbsp;Enter 'This is Bold'<br>
3. &nbsp; &nbsp;Select the word 'Bold'<br>
4. &nbsp; &nbsp;Click the 'B' style button from the standard toolbar.<br>
5. &nbsp; &nbsp;Verify the word 'Bold' appears with Bold style.<br>
6. &nbsp; &nbsp;Enter 'This is Italic'<br>
7. &nbsp; &nbsp;Select the word 'Italic'<br>
8. &nbsp; &nbsp;Click the 'I' style button from the standard toolbar.<br>
9. &nbsp; &nbsp;Verify the word 'Italic' appears with Italic style.<br>
10. &nbsp; &nbsp;Enter This is Orange Color'.<br>
11. &nbsp; &nbsp;Select the word 'Orange'<br>
12. &nbsp; &nbsp;Click the 'Font color' style button from the standard toolbar.<br>
13. &nbsp; &nbsp;Select Orange color.<br>
14. &nbsp; &nbsp;Verify the word 'Orange' appears with Orange color.<br>
15. &nbsp; &nbsp;Press Ctrl+A keys.<br>
16. &nbsp; &nbsp;Verify all the texts are selected.<br>
17. &nbsp; &nbsp;Press Ctrl+C keys. (to copy all the texts)<br>
18. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click Basic Book under Templates.<br>
2. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
3. &nbsp; &nbsp;Verify the book opens in the edit mode.&nbsp;<br>
4. &nbsp; &nbsp;Click the Add page button.<br>
5. &nbsp; &nbsp;Verify the Add page dialog opens.&nbsp;<br>
6. &nbsp; &nbsp;Click the 'Just Text' option under Basic Pages.&nbsp;<br>
7. &nbsp; &nbsp;Click the Add Page button.<br>
8. &nbsp; &nbsp;Verify the page is added in the edit view.<br>
9. &nbsp; &nbsp;Press Ctrl+V keys.<br>
10. &nbsp; &nbsp;Verify the 'Bold' style appears correctly on the first line.<br>
11. &nbsp; &nbsp;Verify the 'Italic'style appears correctly on the second line.&nbsp;<br>
12. &nbsp; &nbsp;Verify the 'Orange' color appears correctly on the third line.&nbsp;</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Close LibreOffice Writer document.<br>
2. &nbsp; &nbsp;Delete the Basic book.&nbsp;<br>
3. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
