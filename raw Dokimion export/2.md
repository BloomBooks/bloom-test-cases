2. &nbsp; &nbsp;Press the Enter Key.<br>
3. &nbsp; &nbsp;Verify Oracle VirtualBox page opens.&nbsp;<br>
4. &nbsp; &nbsp;Click Windows Installer (64-bit) from Windows Platform.<br>
5. &nbsp; &nbsp;Verify the Windows Installer.exe file is downloaded in the Downloads folder.&nbsp;<br>
6. &nbsp; &nbsp;Double click on the Windows Installer.exe file.<br>
7. &nbsp; &nbsp;Verify the Oracle's VM Virtual Box is installed on the system.&nbsp;<br>
8. &nbsp; &nbsp;Type 'https://drive.google.com/drive/folders/1PCYrPcd8E3jnHVJgpjEmnpBeFONZefME' in the browser window(Needs to Developer's approval to enter in this page)<br>
9. &nbsp; &nbsp;Press the Enter key.<br>
10. &nbsp; &nbsp;Click the download icon near &lsquo;IE11-Win7.ova&rsquo; file.<br>
11. &nbsp; &nbsp;Verify the &lsquo;IE11-Win10.ova&rsquo; file downloaded into the Downloads folder.&nbsp;<br>
12. &nbsp; &nbsp;Double click on the Virtual Box icon from the Desktop.<br>
13. &nbsp; &nbsp;Verify the Virtual Box opened on the Desktop.<br>
14. &nbsp; &nbsp;Click the File option.<br>
15. &nbsp; &nbsp;Verify the file menu opens.&nbsp;<br>
16. &nbsp; &nbsp;Click Import Appliance.<br>
17. &nbsp; &nbsp;Verify Import Virtual Appliance window opens.&nbsp;<br>
18. &nbsp; &nbsp;Click the Folder icon near the File field.&nbsp;<br>
19. &nbsp; &nbsp;Verify 'Please choose a virtual appliance file to import' dialog opens.&nbsp;<br>
20. &nbsp; &nbsp;Click IE11-Win7.ova file from the Documents folder.&nbsp;<br>
21. &nbsp; &nbsp;Click the Next button.<br>
22. &nbsp; &nbsp;Verify Appliance Settings dialog opens.&nbsp;<br>
23. &nbsp; &nbsp;Click the Next button.&nbsp;<br>
24. &nbsp; &nbsp;Verify 'Importing appliance..' progress bar appears.&nbsp;<br>
25. &nbsp; &nbsp;Verify 'IE11 &ndash; Win7.ova' appears in the Preview pane.&nbsp;<br>
26. &nbsp; &nbsp;Click the Start button from the Tool bar.<br>
27. &nbsp; &nbsp;Verify Windows 7 OS opens in the Oracle VM VirtualBox.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click the Devices option from the toolbar.&nbsp;<br>
2. &nbsp; &nbsp;Verify the Devices menu opens.&nbsp;<br>
3. &nbsp; &nbsp;Click the Drag and Drop option.<br>
4. &nbsp; &nbsp;Verify a submenu opens.&nbsp;<br>
5. &nbsp; &nbsp;Select Bidirectional option.&nbsp;<br>
6. &nbsp; &nbsp;Copy BloomInstaller.newerversion.exe from the computer, (since Internet Explorer 11 cannot open the bloomlibrary.org website.) (eg., Bloom 6.1.0)<br>
7. &nbsp; &nbsp;Paste the BloomInstaller.anyversion.exe file into IE11-Win7 - Oracle VM VirtualBox &nbsp;<br>
8. &nbsp; &nbsp;Verify the Installer file pasted on the Win 7 - Desktop (VM).<br>
9. &nbsp; &nbsp;Double Click on the Bloom Installer file. (newer version eg., Bloom 6.1.0)<br>
10. &nbsp; &nbsp;Verify a Warning message "This application requires one of the following versions of the .NET Framework: .NETFramework, Version=v4.7.2.. Do you want to install this .NET Framework version now?" with a Yes or No button.&nbsp;<br>
11. &nbsp; &nbsp;Click the Yes button.&nbsp;<br>
12. &nbsp; &nbsp;Verify .NET Framework page opens.&nbsp;<br>
13. &nbsp; &nbsp;Click the 'NET Framework 4.7.2' link under Other versions.<br>
14. &nbsp; &nbsp;Verify the .NET Framework 4.7.2 download page opens.&nbsp;<br>
15. &nbsp; &nbsp;Click the Offline Installer 'Runtime' link<br>
16. &nbsp; &nbsp;Verify the Microsoft .NET Framework page opens.&nbsp;<br>
17. &nbsp; &nbsp;Select 'I have read and accept..' check box.&nbsp;<br>
18. &nbsp; &nbsp;Click the Install button.&nbsp;<br>
19. &nbsp; &nbsp;Verify Installation Is Complete dialog appears.&nbsp;<br>
20. &nbsp; &nbsp;Click the Finish button.<br>
21. &nbsp; &nbsp;Double Click on the Bloom Installer file.&nbsp;<br>
22. &nbsp; &nbsp;Verify a warning message, &ldquo;we are sorry... this version of Bloom requires Windows 10 or greater. As a result you will need to install Bloom 5.4. Bloom will now open a web page where you can download Bloom 5.4"<br>
23. &nbsp; &nbsp;Click the OK button.<br>
24. &nbsp; &nbsp;Verify the bloomlibrary.org web page opens.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete Bloom Installer file.<br>
2. &nbsp; &nbsp;Uninstall .NET 4.7.2 framework.<br>
3. &nbsp; &nbsp;Uninstall Oracle Virtual Box.&nbsp;</p>
<p>&nbsp;</p>
