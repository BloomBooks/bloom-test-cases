# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Remove any files, settings, or data that were left behind by the previous installation or creation of the RTB .<br>
2. &nbsp; &nbsp;Verify an existing .BloomPack file (English Special Template Books'.BloomPack)) appears in the computer. (in Documents folder)<br>
3. &nbsp; &nbsp;Double click on the .BloomPack file (eg., English Special Template Books'.BloomPack). (eg., from the Documents folder)<br>
4. &nbsp; &nbsp;Verify 'Bloom Pack Installation' dialog appears<br>
5. &nbsp; &nbsp;Click the OK button.<br>
6. &nbsp; &nbsp;Verify the Bloom application opens.&nbsp;<br>
7. &nbsp; &nbsp;Verify the RTB book (eg.,'English Special Template Books&rsquo;) appears in the lower left pane of the Collections view.<br>
8. &nbsp; &nbsp;Verify a set of Decodable and Leveled Reader books appears under RTB Book name. (eg., English Special Template Books)</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click on the Decodable Reader book. (eg., Stage 2 Book)<br>
2. &nbsp; &nbsp;Verify MAKE A BOOK USING THIS SOURCE button appears.<br>
3. &nbsp; &nbsp;Click on MAKE A BOOK USING THIS SOURCE button.<br>
4. &nbsp; &nbsp;Verify the book opens in the Edit view.&nbsp;<br>
5. &nbsp; &nbsp;Click Page 1.<br>
6. &nbsp; &nbsp;Verify the sample words appear under the Decodable Reader Tool menu.<br>
7. &nbsp; &nbsp;Verify the decodable settings show the correct stage (eg., Stage 1 of 2) &nbsp;under the Decodable Reader Tool menu.<br>
8. &nbsp; &nbsp;Click on the &lsquo;Setup Stages&rsquo; link.<br>
9. &nbsp; &nbsp;Verify Setup Decodable Reader Tool dialog opens.&nbsp;<br>
10. &nbsp; &nbsp; Verify already added sight words appear on the Stages pane.&nbsp;<br>
11. &nbsp; &nbsp;Click the OK button.&nbsp;<br>
12. &nbsp; &nbsp;Verify Setup Decodable Reader Tool dialog closes.&nbsp;</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the RTB (DR/LR) book.&nbsp;<br>
2. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
