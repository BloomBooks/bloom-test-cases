# ID 109

# Name Automatic Updates Suggestion box

# Description

```html
<p>(20, 21) Check Automatic Updates Suggestion box</p>
```

# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. Uninstall the Bloom application.<br>
2. Goto the windows Folder ".....\AppData\Local\BloomBetaInternal".<br>
3. Delete all the existing folders.</p>
<p>&nbsp;<br>
Action Steps:<br>
1. Download and Install the bloom application on the following location "https://bloomlibrary.org/installersBetaInternal/BloomInstaller.6.2.3087.BetaInternal.exe"<br>
2. Verify the Bloom application launched.<br>
3. Click the Settings button.<br>
4. Verify the Settings dialog appears.<br>
5. Click the Advanced Program Settings tab.<br>
6. Verify the checkbox &nbsp;"Automatically Update Bloom" is selected by default.<br>
7. Verify the checkbox "Team Collection" which is under the Experimental Features appears as Unchecked.<br>
8. Select the "Team Collection" checkbox.<br>
9. Verify the "Team Collection" checkbox becomes checked.<br>
10. Verify the Restart button appears.<br>
11. Click on the Restart button.<br>
12. Verify the bloom gets restarted.<br>
13. Close the Bloom Application.<br>
14. Double click on the bloom short cut icon from the windows Desktop.<br>
15. Verify the Bloom application launched.<br>
16. Click on the "Basic Book" Templates from the Sources For New Books.<br>
17. Verify the "MAKE A BOOK USING THIS SOURCE" button appears on the Right side pane.<br>
18. Click on the "MAKE A BOOK USING THIS SOURCE" button.<br>
19. Verify the Basic Book opens in the Edit Mode.<br>
20. Click on the Front Cover page from the Left pane.<br>
21. Verify the Front Cover page displays to Edit.<br>
22. Click on the Title Box from the Front Cover page<br>
23. Type Title as Test.<br>
24. Leave the Bloom application for 1 minute.<br>
25. Verify the Toast message "Downloading bloom updates" appears on the bottom right corner.<br>
26. Verify the Toast message "Applying updates" appears on the bottom right corner.<br>
27. wait until the Applying updates get completed.<br>
28. Verify the Toast message "Update for "...version number" is ready" appears on the bottom right corner.<br>
29. Verify the "Restart Bloom to Update" link appears on the Toast message.<br>
30. Click on the "Restart Bloom to Update" link.<br>
31. Verify the bloom gets restarted.<br>
32. Click on the Get Help button from the Top Right corner.<br>
33. Verify the "Check For New Version" option appears.<br>
34. Click on the Check For New Version<br>
35. Verify the Toast message "Your Bloom is up to date" appears on the bottom right corner.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. Delete the "Test" book.<br>
2. Close the Bloom application.</p>
<p>&nbsp;</p>
```
