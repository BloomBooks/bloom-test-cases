# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Download a sample picture from the internet. (eg., https://www.freepik.com/free-photos-vectors/sample) in the downloads folder.&nbsp;<br>
2. &nbsp; &nbsp;Rename the picture file as &lsquo;Sample-Picture&amp;Image&rsquo;<br>
3. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click a Basic book.<br>
2. &nbsp; &nbsp;Verify MAKE A BOOK USING THIS SOURCE button.<br>
3. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
4. &nbsp; &nbsp;Verify the book opens in the edit mode.<br>
5. &nbsp; &nbsp;Click the 'Change Image' button on the picture pane.&nbsp;<br>
6. &nbsp; &nbsp;Verify the Image Toolbox dialog opens.&nbsp;<br>
7. &nbsp; &nbsp;Click the File button.<br>
8. &nbsp; &nbsp;Click the Change Image button.<br>
9. &nbsp; &nbsp;Verify Choose Picture File dialog opens.&nbsp;<br>
10. &nbsp; &nbsp;Click the Downloads option.&nbsp;<br>
11. &nbsp; &nbsp;Verify the Downloads folder opens on the right pane.&nbsp;<br>
12. &nbsp; &nbsp;Select the Sample picture.&nbsp;<br>
13. &nbsp; &nbsp;Click the Open button.&nbsp;<br>
14. &nbsp; &nbsp;Verify the picture appears on the preview pane.&nbsp;<br>
15. &nbsp; &nbsp;Click the OK button.&nbsp;<br>
16. &nbsp; &nbsp;Verify the Picture is imported into the Front cover page.&nbsp;<br>
17. &nbsp; &nbsp;Click the Publish tab.<br>
18. &nbsp; &nbsp;Click the PDF &amp; Print button.<br>
19. &nbsp; &nbsp;Verify Publish to PDF &amp; Print view opens.<br>
20. &nbsp; &nbsp;Click the Simple mode under Booklet Mode.<br>
21. &nbsp; &nbsp;Verify the PDF book is created.&nbsp;<br>
22. &nbsp; &nbsp;Verify the image appears correctly on the Front cover page.&nbsp;<br>
23. &nbsp; &nbsp;Click the Close button.<br>
24. &nbsp; &nbsp;Verify the Bloom closes.&nbsp;<br>
25. &nbsp; &nbsp;Restart Bloom application.&nbsp;<br>
26. &nbsp; &nbsp;Click the EDIT THIS BOOK button.<br>
27. &nbsp; &nbsp;Verify the image appears correctly on the Front cover page.&nbsp;</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the Sample Picture.<br>
2. &nbsp; &nbsp;Delete the Basic book.<br>
3. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
