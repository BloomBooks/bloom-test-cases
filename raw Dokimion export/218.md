# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Removed Bloom folder from C:\Users\UserName\AppData\Local<br>
2. &nbsp; &nbsp;Remove Bloom folder from C:\Users\UserName\AppData\Local\SIL<br>
3. &nbsp; &nbsp;Restart Windows<br>
4. &nbsp; &nbsp;Download the latest version of BloomInstaller.exe file (eg.,BloomInstaller.x.x.xxxx.xxx) from Bloomlibrary.org site.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Double click on the latest version of BloomInstaller.exe file.<br>
2. &nbsp; &nbsp;Verify Bloom x.x installation dialog opens.<br>
3. &nbsp; &nbsp;Click the 'I accept the terms of the license agreement' button.<br>
4. &nbsp; &nbsp;Verify 'Open/Create Collections' dialog opens.<br>
5. &nbsp; &nbsp;Click the UI language pulldown menu.<br>
6. &nbsp; &nbsp;Click 'Tamil' language.<br>
7. &nbsp; &nbsp;Verify the Tamil UI language appears in the 'Open/Create Collections' dialog.<br>
8. &nbsp; &nbsp;Click the 'Open/Create collections' button which appears in Tamil UI.<br>
9. &nbsp; &nbsp;Verify the Choose Language dialog appears in Tamil UI.<br>
10. &nbsp; &nbsp;Enter'ta' in the Search box.<br>
11. &nbsp; &nbsp;Verify Tamil language bar appears.<br>
12. &nbsp; &nbsp;Click the 'Tamil' language bar.<br>
13. &nbsp; &nbsp;Verify 'Tamil' script bar appears.<br>
14. &nbsp; &nbsp;Click the 'Tamil' script bar.&nbsp;<br>
15. &nbsp; &nbsp;Click the 'Next' button in Tamil UI.<br>
16. &nbsp; &nbsp;Verify Font and Script dialog appears.<br>
17. &nbsp; &nbsp;Click the Next button.<br>
18. &nbsp; &nbsp;Verify the Give language location dialog appears in Tamil UI.<br>
19. &nbsp; &nbsp;Click the Next button in Tamil UI.<br>
20. &nbsp; &nbsp;Verify the Tamil Book Collection Name name appears as the Book Collection name.&nbsp;<br>
21. &nbsp; &nbsp;Click the Next button in Tamil UI.<br>
22. &nbsp; &nbsp;Verify Ready to Create New Collection (appears in Tamil UI) dialog opens.<br>
23. &nbsp; &nbsp;Click the Finish button.<br>
24. &nbsp; &nbsp;Verify the Bloom application is restarted.<br>
25. &nbsp; &nbsp;Verify the Bloom application appears with Tamil UI.<br>
26. &nbsp; &nbsp;Verify the Software Updates dialog opens.<br>
27. &nbsp; &nbsp;Verify the Software Updates dialog appears in the Tamil UI.<br>
28. &nbsp; &nbsp;Click 'Just let me know about the new version' radio button.<br>
29. &nbsp; &nbsp;Click the OK button.<br>
30. &nbsp; &nbsp;Click the Help button.<br>
31. &nbsp; &nbsp;Verify the UI language appears in Tamil.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the Tamil Book Collection.<br>
2. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
