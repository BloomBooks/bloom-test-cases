# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;With a fresh machine go to C:/Users/&lt;name&gt;/AppData/Local/SIL/Bloom. &nbsp;<br>
2. &nbsp; &nbsp;Remove all directories that start with X.X. &nbsp; (eg., 5.4, 6.2)<br>
3. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Verify Bloom Notes &amp; License dialog appears.&nbsp;<br>
2. &nbsp; &nbsp;Click 'I accept the terms of the..' button.<br>
3. &nbsp; &nbsp;Verify Open/Create Collections dialog opens.&nbsp;<br>
4. &nbsp; &nbsp;Select an existing book collection. (eg., English Book Collection)<br>
5. &nbsp; &nbsp;Verify 'Software Updates' dialog appears.&nbsp;<br>
6. &nbsp; &nbsp;Select 'Just let me know about the new version' option.<br>
7. &nbsp; &nbsp;Click the OK button.<br>
8. &nbsp; &nbsp;Verify there is no 'Bloom Community Forum' dialog appears.&nbsp;<br>
9. &nbsp; &nbsp;Click the Close button.<br>
10. &nbsp; &nbsp;Verify the Bloom application closes.&nbsp;<br>
11. &nbsp; &nbsp;go to C:\Users\UserName\AppData\Local\SIL\BloomX\VersionNumber<br>
12. &nbsp; &nbsp;Verify user config file appears.<br>
13. &nbsp; &nbsp;Open it in the Notepad application.<br>
14. &nbsp; &nbsp;Verify the user.config file opens in the notepad application.<br>
15. &nbsp; &nbsp;Press Ctrl+F keys.<br>
16. &nbsp; &nbsp;Enter 'Forum' in the Find dialog.&nbsp;<br>
17. &nbsp; &nbsp;Press the Enter key.&nbsp;<br>
18. &nbsp; &nbsp;Verify 'ForumInvitationLastShown' name appears.<br>
19. &nbsp; &nbsp;Verify the value of the setting "ForumInvitationLastShown" is set to the current date.&nbsp;<br>
20. &nbsp; &nbsp;Set the date value to (two weeks before date)<br>
21. &nbsp; &nbsp;Press the Enter key.<br>
22. &nbsp; &nbsp;Verify 'ForumInvitationAcknowledged' name appears.<br>
23. &nbsp; &nbsp;Set the value to 'False' (If it is not set to False)<br>
24. &nbsp; &nbsp;Save the file.&nbsp;<br>
25. &nbsp; &nbsp;Launch Bloom application.<br>
26. &nbsp; &nbsp;Verify "Bloom Community Forum" dialog appears on the Bloom Editor.&nbsp;</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Go to C:/Users/&lt;name&gt;/AppData/Local/SIL/Bloom.<br>
2. &nbsp; &nbsp;Remove all directories that start with X.X. &nbsp; (eg., 5.4, 6.2)<br>
3. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
