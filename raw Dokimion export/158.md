4. &nbsp; &nbsp;Right Click on the .zip file.<br>

6. &nbsp; &nbsp;Click the 'Extract here' option.<br>
7. &nbsp; &nbsp;Verify the DoulosSIL folder appears.<br>

10. &nbsp; &nbsp;Double click on the DoulosSIL-Regular ttf file.<br>
11. &nbsp; &nbsp;Verify the Install dialog opens.<br>
12. &nbsp; &nbsp;Click the Install button.<br>

18. &nbsp; &nbsp;Verify the Settings dialog opens.&nbsp;<br>
19. &nbsp; &nbsp;Click the Bloom Subscription tab.<br>
20. &nbsp; &nbsp;Verify the Bloom Subscription tab opens.&nbsp;<br>
21. &nbsp; &nbsp;Enter 'Test-727011-1339' in the Subscription code.&nbsp;<br>
22. &nbsp; &nbsp;Verify the Restart button appears instead of the OK button.<br>
23. &nbsp; &nbsp;Click the Restart button.<br>
24. &nbsp; &nbsp;Verify Bloom gets restarted.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click an eBook under Templates in the Collections view.<br>
2. &nbsp; &nbsp;Verify MAKE A BOOK USING THIS SOURCE button appears on the preview pane.&nbsp;<br>
3. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
4. &nbsp; &nbsp;Verify the book opens in the Edit mode.&nbsp;<br>
5. &nbsp; &nbsp;Enter 'New Fonts in Bloom Reader' on the Front cover title box.<br>

17. &nbsp; &nbsp;Click the gear icon (which appears in the bottom left of the text box)<br>

19. &nbsp; &nbsp;Click the 'Character' tab.<br>
