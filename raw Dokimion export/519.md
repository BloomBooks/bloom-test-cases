# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. Launch Bloom application.<br>
2. Open a collection with an enterprise subscription.</p>
<p>&nbsp;</p>
<p>Action Steps<br>
1. &nbsp; &nbsp;Click on the "Basic Book" from the Sources For New Books in the left Pane.<br>
2. &nbsp; &nbsp;Verify that the MAKE A BOOK USING THIS SOURCE button appears in the right pane.<br>
3. &nbsp; &nbsp;Click the MAKE A BOOK USING THIS SOURCE button.<br>
4. &nbsp; &nbsp;Verify that the Basic Book opens in Edit Mode.<br>
5. &nbsp; &nbsp;Click the Add Page + button.<br>
6. &nbsp; &nbsp;Verify that the Add Page&hellip; dialog appears.<br>
7. &nbsp; &nbsp;Select the Just a Picture template from Basic Pages.<br>
8. &nbsp; &nbsp;Click ADD PAGE.<br>
9. &nbsp; &nbsp;Verify that the Just a Picture page is displayed in the right pane.<br>
10. &nbsp; &nbsp;Verify that an Image Box is present on Page 1.<br>
11. &nbsp; &nbsp;Verify that the Change Image button appears on Page 1.<br>
12. &nbsp; &nbsp;Click the Hamburger menu (top-right corner).<br>
13. &nbsp; &nbsp;Verify that the Talking Book Tool appears.<br>
14. &nbsp; &nbsp;Click the More&hellip; button.<br>
15. &nbsp; &nbsp;Verify that the Overlay Tool checkbox is unchecked by default.<br>
16. &nbsp; &nbsp;Select the Overlay Tool checkbox.<br>
17. &nbsp; &nbsp;Verify that the Overlay Tool section expands.<br>
18. &nbsp; &nbsp;Verify that the video icon button is displayed in the Overlay Tool.<br>
19. &nbsp; &nbsp;Drag and drop video icon from the Overlay Tool onto Page 1.<br>
20. &nbsp; &nbsp;Verify that a new empty video overlay appears on Page 1.<br>
21. &nbsp; &nbsp;Click the new empty video overlay.<br>
22. &nbsp; &nbsp;Verify that the Record Yourself button appears below the video overlay.<br>
23. &nbsp; &nbsp;Verify that the Image Box and Change Image button disappear from Page 1.<br>
24. &nbsp; &nbsp;Click the Record Yourself button.<br>
25. &nbsp; &nbsp;Verify that a camera permission alert appears (only the first time).<br>
26. &nbsp; &nbsp;Click Allow.<br>
27. &nbsp; &nbsp;Verify that the alert disappears.<br>
28. &nbsp; &nbsp;Verify that the Sign Language Tool appears on the right pane.<br>
29. &nbsp; &nbsp;Verify that the camera feed is active in the Sign Language Tool.<br>
30. &nbsp; &nbsp;Verify that the Red Record button is displayed.<br>
31. &nbsp; &nbsp;Click the Red Record button.<br>
32. &nbsp; &nbsp;Verify that recording begins.<br>
33. &nbsp; &nbsp;Record the video.<br>
34. &nbsp; &nbsp;Click the Stop button.<br>
35. &nbsp; &nbsp;Verify that recording stops.<br>
36. &nbsp; &nbsp;Verify that the recorded video appears in the Page 1 video overlay.<br>
37. &nbsp; &nbsp;Click the Play button on the video overlay.<br>
38. &nbsp; &nbsp;Verify that the video starts playing.<br>
39. &nbsp; &nbsp;Click the Pause button.<br>
40. &nbsp; &nbsp;Verify that the video pauses playing.<br>
41. &nbsp; &nbsp;Click the Overlay Tool in the right pane again.<br>
42. &nbsp; &nbsp;Verify that the Overlay Tool section expands.<br>
43. &nbsp; &nbsp;Drag and drop another video icon onto Page 1.<br>
44. &nbsp; &nbsp;Verify that empty video overlay is added to Page 1.<br>
45. &nbsp; &nbsp;Click the new empty video overlay.<br>
46. &nbsp; &nbsp;Click the already recorded video overlay.<br>
47. &nbsp; &nbsp;Verify that the already recorded video does not play when it is selected.</p>
<p><br>
Cleanup Steps:<br>
1. Delete the "Test" book.<br>
2. Close the Bloom application.</p>
```
