## Step

### Action

```html
<p>Setup Steps:</p>
<p><br>
1. &nbsp; &nbsp;Launch Bloom application<br>
&nbsp;<br>
Action Steps:</p>
<p><br>
1. Click the "Basic Book".<br>
2. Verify that the book preview is displayed in the right pane.<br>
3. Click the &ldquo;MAKE A BOOK USING THIS SOURCE&rdquo; button.<br>
4. Verify that &ldquo;Basic Book&rdquo; opens in edit mode.<br>
5. Click the "Front cover" page.<br>
6. Type "Sample Book" in the "BOOK title in English" field.<br>
7. Hover the mouse over the picture icon in the "Front cover" page.<br>
8. Verify that the "Change image" button is displayed.<br>
9. Click the "Change image' button in the picture pane.<br>
10. Verify the "Image Toolbox" dialog box opens.<br>
11. Click the File button.<br>
12. Verify "Choose Picture Files" dialog box opens.&nbsp;<br>
13. Click the valid ".png" image from the "Choose Picture Files" dialog box.<br>
14. Click the Open Key.<br>
15. Verify that the selected image appears on the "Image Toolbox" dialog box.<br>
16. Click the OK button.<br>
17. Verify that the selected picture appears in the Front cover page.<br>
18. Navigate to the "Collection" tab.<br>
19. Verify that the "Sample Book" is displayed under the "English Books" area.<br>
20. Right click the "Sample Book".<br>
21. Select the "Show in File Explorer" option from the submenu.<br>
22. Verify that the folder named "Sample Book" is displayed.<br>
23. Double-click the folder named "Sample Book" to open it.<br>
24. Verify that the "Sample Book.html" file is displayed.<br>
25. Open the "Sample Book.html" file with notepad.<br>
26. Verify that the "Sample Book" file is opened in the notepad.<br>
27. Verify that there should be no instance of "ckeditor or other user interface leftovers" should be found.<br>
&nbsp;<br>
Cleanup Steps:</p>
<p>1. Delete the "Sample Book".<br>
2. Close Bloom application.</p>
<p>&nbsp;</p>
```
