### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Type "https://dev.bloomlibrary.org/" in the browser window.<br>
2. &nbsp; &nbsp;Press the Enter key.<br>
3. &nbsp; &nbsp;Verify dev.bloomlibrary page opens.&nbsp;<br>
4. &nbsp; &nbsp;Click the Sign In /Sign Up option.<br>
5. &nbsp; &nbsp;Verify Sign In /Sign Up dialog opens.&nbsp;<br>
6. &nbsp; &nbsp;Click Sign in with an email button.<br>
7. &nbsp; &nbsp;Verify Sign In /Sign Up dialog opens.<br>
8. &nbsp; &nbsp;Enter a valid email address in the Email box. (Use Collector account credentials)<br>
9. &nbsp; &nbsp;Click the Next button.<br>
10. &nbsp; &nbsp;Enter Password in the Password box.&nbsp;<br>
11. &nbsp; &nbsp;Click the Sign In button.<br>
12. &nbsp; &nbsp;Verify you are logged in as a Collector in dev.bloomlibrary.org. web page.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click PRATHAM books.&nbsp;<br>
2. &nbsp; &nbsp;Verify Pratham books page opens.&nbsp;<br>
3. &nbsp; &nbsp;Click any book under Pratham books. (eg., How do Airplanes Fly?)<br>
4. &nbsp; &nbsp;Verify the book opens on the web page.&nbsp;<br>
5. &nbsp; &nbsp;Click the "DOWNLOAD INTO BLOOM FOR EDITING" button.<br>
6. &nbsp; &nbsp;Verify a new instance of bloom opens.<br>
7. &nbsp; &nbsp;Verify the thumbnail should display the downloaded badge under "Books From BloomLibrary.org" heading. (which appears on the lower left pane of the Bloom application)<br>
8. &nbsp; &nbsp;Verify the "sources for new books" should be hidden.&nbsp;<br>
9. &nbsp; &nbsp;Click the Settings button.<br>
10. &nbsp; &nbsp;Verify the Settings dialog opens.&nbsp;<br>
11. &nbsp; &nbsp;Click the Bloom Subscription tab.<br>
12. &nbsp; &nbsp;Verify 'This collection is in "Download for Edit" mode. The book has the same subscription setting as when it was las uploaded' message appears.&nbsp;<br>
13. &nbsp; &nbsp;Click the OK button.&nbsp;<br>
14. &nbsp; &nbsp;Verify the name of the collection should be "From Bloom Library-NAME OF BOOK" (eg., From Bloom Library - How Do Aeroplanes Fly)<br>
15. &nbsp; &nbsp;Click the EDIT THIS BOOK button.<br>
16. &nbsp; &nbsp;Verify the book opens in the edit view.&nbsp;<br>
17. &nbsp; &nbsp;Click the Credits Page.<br>
18. &nbsp; &nbsp;Verify the Copyright shows the branding name.&nbsp;<br>
19. &nbsp; &nbsp;Click the Collections tab.<br>
20. &nbsp; &nbsp;Verify the Collections view opens.&nbsp;<br>
21. &nbsp; &nbsp;Verify there is no other option to create a new book in this collection.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Logged out from dev.bloomlibrary.org.<br>
2. &nbsp; &nbsp;Delete the downloaded book.<br>
3. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
