# Steps

## Step

### Action

```html
<p>Setups Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click a Basic book.<br>
2. &nbsp; &nbsp;Verify &lsquo;MAKE A BOOK USING THIS SOURCE&rsquo; button is available on the right pane.&nbsp;<br>
3. &nbsp; &nbsp;Click the Close button.<br>
4. &nbsp; &nbsp;Verify the Bloom closes.&nbsp;<br>
5. &nbsp; &nbsp;Open a browser window.<br>
6. &nbsp; &nbsp;Enter &lsquo;https://bloomlibrary.org/language:en/book/b9LEDelBFg?lang=en&rsquo; in a Search bar.&nbsp;<br>
7. &nbsp; &nbsp;Press the Enter key.<br>
8. &nbsp; &nbsp;Verify the book (eg., Wild Animals)opens on the page.&nbsp;<br>
9. &nbsp; &nbsp;Click the 'Translate into your language' - Download into Bloom Editor' button.<br>
10. &nbsp; &nbsp;Verify Downloading book progress dialog appears.<br>
11. &nbsp; &nbsp;Verify Download complete dialog appears with OK button.&nbsp;<br>
12. &nbsp; &nbsp;Click the OK button.<br>
13. &nbsp; &nbsp;Verify the 'Wild Animals' book opens in the Bloom application.<br>
14. &nbsp; &nbsp;Verify MAKE A BOOK USING THIS SOURCE button is available for translation.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the Basic book.<br>
2. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
