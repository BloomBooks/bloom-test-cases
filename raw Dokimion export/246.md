# Steps

## Step

### Action

```html
<p>Setup Steps:<br>
1. &nbsp; &nbsp;Launch Bloom application.</p>
<p>&nbsp;</p>
<p>Action Steps:<br>
1. &nbsp; &nbsp;Click a Basic book.<br>
2. &nbsp; &nbsp;Verify MAKE A BOOK USING THIS SOURCE button.<br>
3. &nbsp; &nbsp;Click MAKE A BOOK USING THIS SOURCE button.<br>
4. &nbsp; &nbsp;Verify the book opens in the edit mode.<br>
5. &nbsp; &nbsp;Enter 'The Golden Touch' in the Front Cover title box.&nbsp;<br>
6. &nbsp; &nbsp;Click the 'Change Image' button on the picture pane.&nbsp;<br>
7. &nbsp; &nbsp;Verify the Image Toolbox dialog opens.&nbsp;<br>
8. &nbsp; &nbsp;Enter 'Car' in the Search box.<br>
9. &nbsp; &nbsp;Press the Enter key.<br>
10. &nbsp; &nbsp;Verify Car images appear on the preview pane.&nbsp;<br>
11. &nbsp; &nbsp;Click any of the Car images.<br>
12. &nbsp; &nbsp;Click the OK button.&nbsp;<br>
13. &nbsp; &nbsp;Verify the Car image appears on the picture pane.<br>
14. &nbsp; &nbsp;Click the Collections tab.<br>
15. &nbsp; &nbsp;Verify the Collections view opens.&nbsp;<br>
16. &nbsp; &nbsp;Right click on the 'The Golden Touch' book thumbnail.<br>
17. &nbsp; &nbsp;Verify a Context menu appears.<br>
18. &nbsp; &nbsp;Click the 'Show in File Explorer' option.&nbsp;<br>
19. &nbsp; &nbsp;Verify the book (eg., The Golden Touch) directory opens. (eg., C:\Users\UserName\Documents\Bloom\English Books\The Golden Touch)<br>
20. &nbsp; &nbsp;Double click on the book'The Golden Touch' folder.&nbsp;<br>
21. &nbsp; &nbsp;Verify the book folder opens.<br>
22. &nbsp; &nbsp;Delete the Front cover PNG file. (eg., aor_jeep-1.PNG)<br>
23. &nbsp; &nbsp;Close the window.&nbsp;<br>
24. &nbsp; &nbsp;Close Bloom application.&nbsp;<br>
25. &nbsp; &nbsp;Restart Bloom.&nbsp;<br>
26. &nbsp; &nbsp;Click the EDIT THIS BOOK button.<br>
27. &nbsp; &nbsp;Verify the 'Car' image is missing in the Front cover page.&nbsp;<br>
28. &nbsp; &nbsp;Verify an error message 'This picture, aor_jeep-1.png, is missing or was loading too slowly' appears in the front cover page.</p>
<p>&nbsp;</p>
<p>Cleanup Steps:<br>
1. &nbsp; &nbsp;Delete the Basic book.&nbsp;<br>
2. &nbsp; &nbsp;Close Bloom application.</p>
<p>&nbsp;</p>
```
