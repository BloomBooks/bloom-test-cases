2. &nbsp; &nbsp;Verify a popup menu opens.&nbsp;<br>
3. &nbsp; &nbsp;Click the "Check For New Version" option.<br>
4. &nbsp; &nbsp;Verify "Your Bloom is up to date" toast message appears.<br>
5. &nbsp; &nbsp;Enter 'Network ' in the Windows Search bar.&nbsp;<br>

28. &nbsp; &nbsp;Click the 'Enable' option.&nbsp;<br>
29. &nbsp; &nbsp;Verify the Network connection is enabled.&nbsp;<br>
30. &nbsp; &nbsp;Verify "Can't check for new version offline" message should not continue after reconnecting the Internet.</p>
