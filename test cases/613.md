# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Other Collection button.
2. Verify Open/Create Collections dialog opens.
3. Click the 'Create New Collection' button.
4. Verify 'Create New Bloom Collection' dialog box opens.
5. Type "foo bar" in the Search bar.
6. Verify that Text under "Create Unlisted Language" is "qaa-x-foobar".
7. Delete the text in the Search bar.
8. Type " 1 23 456 7890-@({}=) é, ñ, hi123there " in the Search bar.
9. Verify the Text under "Create Unlisted Language" is "qaa-x-12345678-hi123the"
10. Delete the text in the Search bar.
11. Type "hi-there-12-5 6-12345@{}(* 6789" in the Search bar.
12. Verify the Text under "Create Unlisted Language" is "qaa-x-hi-there-12-56-12345678"
13. Click "Create Unlisted Language".
14. Verify the Unlisted Language Tag dialog opens.
15. Verify the Text in Name text box is " hi-there-12-5 6-12345@{}(* 6789 ".
16. Scroll down the dialog box.
17. Verify the Text to the right of "BCP 47 Tag:" is "qaa-x-hi-there-12-56-12345678".
18. Click the CANCEL button.
19. Verify the Unlisted Language Tag dialog closes.
20. Click the CANCEL button.
21. Verify 'Create New Bloom Collection' dialog box closes.
22. Click the CLOSE button.
23. Verify Open/Create Collections dialog closes.

**Cleanup Steps**

1. Close Bloom application.
