# Steps

**Setup Steps**

1. Download the problem book from the given link : https://s3.amazonaws.com/bloom-problem-books/BL-10324.zip
2. Extract the zip file into a New Folder.
3. Double click on “Kanuri, Manga Books” Bloom book collection
4. Verify the book opens in the Bloom application.

**Action Steps**

1. Click on the 'Jumpy Decodable' book thumbnail.
2. Click the EDIT THIS BOOK button.
3. Verify the book opens in the Edit view.
4. Enter 'whatuu' in the text pane.
5. Verify the cursor should not jump when typing on the text pane.

**Cleanup Steps**

1. Delete the Jumpy Decodable Reader book.
2. Close Bloom application.
