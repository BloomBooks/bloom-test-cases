# Steps

**Setups Steps**

1. 3 book collections: A, B and C with several books in each.
2. Several downloaded books should be there under "Books from BloomLibrary.org".(which appears at the lower left pane in Collections view)
3. Open Book Collection B in the bloom application.
4. Add many books using the "SHOW ANOTHER COLLECTION" button.
5. Relaunch Bloom application.

**Action Steps**

1. Click the 'Other Collection' button.
2. Verify Open/Create Collections dialog opens.
3. Click Collection A.
4. Verify Collection A opens.
5. Select a book in its collection.
6. Click the 'Other Collection' button.
7. Verify Open/Create Collections dialog opens.
8. Click Collection B.
9. Verify Collection B opens.
10. Verify no book is selected.
11. Select a book in Collection B.(Under Get More Source Books at Bloomlibrary.org)
12. Click the 'Other Collection' button.
13. Verify Open/Create Collections dialog opens.
14. Click Collection A.
15. Verify Collection A opens.
16. Verify: book B is preselected (shows in Preview panel [right] and is selected under book sources [bottom left panel]).
17. Click the 'Other Collection' button.
18. Verify Open/Create Collections dialog opens.
19. Click Collection C.
20. Verify Collection C opens.
21. Verify: book B continues to be preselected in preview.
22. Select any book (eg., 08. God Saves Joseph and His Family) in Books from BloomLibrary.org.
23. Click the 'Other Collection' button.
24. Verify Open/Create Collections dialog opens.
25. Click Collection A.
26. Verify Collection A opens.
27. Verify the book (eg., eg., 08. God Saves Joseph and His Family) is preselected.
28. Select any Template (e.g. Arithmetic Template).
29. Click the 'Other Collection' button.
30. Verify Open/Create Collections dialog opens.
31. Click Collection C.
32. Verify Collection C opens.
33. Verify the template (eg., Arithmetic Template) is preselected.
34. Select a book in collection C.
35. Click the 'Other Collection' button.
36. Verify Open/Create Collections dialog opens.
37. Click Collection A.
38. Verify Collection A opens.
39. Verify no book is selected.

**Cleanup Steps**

1. Delete book Collection A.
2. Delete book Collection B.
3. Delete book Collection C.
4. Close Bloom application.
