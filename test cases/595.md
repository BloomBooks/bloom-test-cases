# Steps

**Setup Steps**

1. Connect the Android Mobile (with Bloom Reader installed) to the computer.
2. Download the “I am sick” book from the given link: https://bloomlibrary.org/chetana/book/GrFisHU9Jc?lang=en
3. Copy the "la+niña.bloomPUB' file from the computer.
4. Paste the ‘la+niña.bloomPUB’ file into the Mobile's folder. (e.g., Internal Storage)
5. Launch Bloom Reader application in the Android Mobile.

**Action Steps**

1. Tap the Hamburger button.
2. Verify the file menu opens.
3. Tap the 'BloomPUB file' option.
4. Verify the “la+niña.bloomPUB” file appears in the Mobile's folder.
5. Tap the "la+niña.bloomPUB' file.
6. Verify the book opens in the Bloom player.
7. Tap the 'SIL.ORG Back Cover' blue link button.
8. Verify browser app displays the corresponding (eg., asia.sil.org) page.

**Cleanup Steps**

1. Disconnect the Android Mobile from the computer.
2. Delete the ‘la+niña.bloomPUB' file.
3. Close Bloom application.
