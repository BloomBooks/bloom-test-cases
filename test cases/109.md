# Steps

**Setup Steps**

1. Uninstall the Bloom application.
2. Goto the windows Folder ".....\AppData\Local\BloomBetaInternal".
3. Delete all the existing folders.

**Action Steps**

1. Download and Install the bloom application on the following location "https://bloomlibrary.org/installersBetaInternal/BloomInstaller.6.2.3087.BetaInternal.exe"
2. Verify the Bloom application launched.
3. Click the Settings button.
4. Verify the Settings dialog appears.
5. Click the Advanced Program Settings tab.
6. Verify the checkbox  "Automatically Update Bloom" is selected by default.
7. Verify the checkbox "Team Collection" which is under the Experimental Features appears as Unchecked.
8. Select the "Team Collection" checkbox.
9. Verify the "Team Collection" checkbox becomes checked.
10. Verify the Restart button appears.
11. Click on the Restart button.
12. Verify the bloom gets restarted.
13. Close the Bloom Application.
14. Double click on the bloom short cut icon from the windows Desktop.
15. Verify the Bloom application launched.
16. Click on the "Basic Book" Templates from the Sources For New Books.
17. Verify the "MAKE A BOOK USING THIS SOURCE" button appears on the Right side pane.
18. Click on the "MAKE A BOOK USING THIS SOURCE" button.
19. Verify the Basic Book opens in the Edit Mode.
20. Click on the Front Cover page from the Left pane.
21. Verify the Front Cover page displays to Edit.
22. Click on the Title Box from the Front Cover page
23. Type Title as Test.
24. Leave the Bloom application for 1 minute.
25. Verify the Toast message "Downloading bloom updates" appears on the bottom right corner.
26. Verify the Toast message "Applying updates" appears on the bottom right corner.
27. wait until the Applying updates get completed.
28. Verify the Toast message "Update for "...version number" is ready" appears on the bottom right corner.
29. Verify the "Restart Bloom to Update" link appears on the Toast message.
30. Click on the "Restart Bloom to Update" link.
31. Verify the bloom gets restarted.
32. Click on the Get Help button from the Top Right corner.
33. Verify the "Check For New Version" option appears.
34. Click on the Check For New Version
35. Verify the Toast message "Your Bloom is up to date" appears on the bottom right corner.

**Cleanup Steps**

1. Delete the "Test" book.
2. Close the Bloom application.
