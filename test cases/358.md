# Steps

**Setup Steps**

1. Updated Internet Explorer Version should be available.
2. Download Bloom 6.1 Beta Internal build from bloomlibrary.org.

**Action Steps**

1. Double click on the Bloom Installer icon.
2. Verify a message like 'Install dotnet version 4.8 ..' appears.
3. Click the OK button to install.
4. Verify the Installation window flashes on the screen.
5. Verify Bloom application window opens.

**Cleanup Steps**

1. Close Bloom application.
