# Steps

**Setup Steps**

1. Download a sample picture from the given link: https://pixabay.com/photos/windows-sample-net-door-7509369/
2. Launch Bloom application.

**Action Steps**

1. Click a Basic book.
2. Click MAKE A BOOK USING THIS SOURCE button.
3. Verify the book opens in the Edit mode.
4. Enter ‘Test Book’ in the Front cover title box.
5. Click the Change Image button.
6. Verify Image Toolbox dialog opens.
7. Click the File button.
8. Verify Choose Picture File dialog box appears.
9. Click the Downloads folder. (from the left pane)
10. Verify the Downloads folder opens.
11. Select the Sample picture.
12. Click the Open button.
13. Verify the Sample picture appears on the Preview pane.
14. Click the OK button.
15. Verify the image appears on the Picture pane.
16. Click the Copyright button.
17. Verify the Copyright and License dialog opens.
18. Enter 'Tester' in the Illustrator/Photographer text box.
19. Enter 'Tester One' in the Copyright Holder text box.
20. Click the OK button.
21. Verify 'Picture Intellectual property Information' dialog appears.
22. Click the Yes button.
23. Click the Title page.
24. Verify 'Paste Image Credits' link button.
25. Verify Image credits appears on the page.
26. Verify the URL for pixabay should not appear on the credits Information.

**Cleanup Steps**

1. Delete the Basic book.
2. Close Bloom application.
