# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Template Starter book.
2. Click 'Make a book using this Source' button available on the right pane.
3. Click the 'Make a book using this Source' button.
4. Verify it opens the edit mode.
5. Click the Add page button.
6. Verify the Add page dialog opens.
7. Click the Custom template page.
8. Click the Add page button.
9. Verify the Custom template is added.
10. Click the Picture link button from the Custom page.
11. Verify the Picture pane appears.
12. Click the + button.
13. Click the Text link button.
14. Verify the Text pane is added to the page.
15. Turn off the Change Layout toggle switch.
16. Verify the Custom page appears with Picture and text pane.
17. Verify the language pull down menu is disabled.
18. Click the Custom label name.
19. Enter 'New Template' in the Custom option.
20. Verify the typed text appears in the local language.
21. Verify the custom thumbnail shows up in the under pages pane.

**Cleanup Steps**

1. Delete the Template Starter book.
2. Close Bloom application.
