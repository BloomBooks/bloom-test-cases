# Steps

**Setup Steps**

1. Import image from the given link : https://www.pexels.com/photo/person-with-body-painting-1209843/ (into downloads folder.)
2. Right click on the image (eg., pexels-mccutcheon-1209843.jpg).
3. Verify popup menu appears.
4. Click the Properties option.
5. Verify ‘pexels-mccutcheon-1209843 properties’ dialog opens.
6. Click the Details tab.
7. Verify the image dimension. (eg., 5616x3744)
8. Launch Bloom application.

**Action Steps**

1. Click the Basic Book  under Templates in the Collections view.
2. Verify MAKE A BOOK USING THIS SOURCE button appears on the preview pane.
3. Click MAKE A BOOK USING THIS SOURCE button.
4. Verify the book opens in the Edit mode.
5. Click the Change Image button in the Front Cover page.
6. Verify the Image toolbox opens.
7. Click the File button.
8. Verify Choose Picture File dialog opens.
9. Click the Downloads option.
10. Verify the Downloads folder opens.
11. Select the image (eg., pexels-mccutcheon-1209843.jpg).
12. Click the Open button.
13. Verify the image appears in the Preview pane.
14. Click the OK button.
15. Verify the image appears in the picture pane.
16. Click on the image.
17. Hover the mouse pointer over the image.
18. Verify an information dialog box appears.
19. Verify the image shrunk to 469x579 dots.

**Cleanup Steps**

1. Delete the Basic book.
2. Close Bloom application.
