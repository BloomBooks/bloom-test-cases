# Steps

**Setup Steps**

1. Unzip attached collection from the given link: https://issues.bloomlibrary.org/youtrack/issue/BL-8019
2. Copy the .bloomCollection file. (eg., BL-7993 TESTING)
3. Paste the .bloomCollection file in a different location. (eg., paste it in D:\ drive)

**Action Steps**

1. Double click on the .bloomCollection file. (eg., BL-7993 TESTING)
2. Verify 'Bloom had a Problem' displays.
3. Verify 'Bloom book folders must have only one file ending..' message appears.
4. Click on the Close button.
5. Verify another error message "Bloom book folder must have only one file ending in .htm. In the book .." appears.
6. Click the Close button.
7. Verify Broken thumbnail appears under Book Collections.

**Cleanup Steps**

1. Delete the corrupted bloom collection.
2. Close Bloose application.
