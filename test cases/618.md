# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the 'Other Collection' button.
2. Verify 'Open/Create Collections' dialog opens.
3. Click the 'Create New Collection' button.
4. Verify 'Create New Bloom Collection'  dialog opens.
5. Enter 'zsm' in the Search box.
6. Verify 'Bahasa Malaysia' language bar appears.
7. Click 'Bahasa Malaysia' language bar.
8. Verify 'Latin' script box appears.
9. Click the 'Latin' script box.
10. Verify name Display name should be "Bahasa Malaysia"
11. Verify the code "ms" appears under the Customize label.
12. Click the Next button.
13. Verify Font and Script view appears.
14. Click the Next button.
15. Verify Give Language Location view appears.
16. Click the Next button.
17. Verify 'Bahasa Malaysia Books' appear in the Collection Name box.
18. Click the Next button.
19. Verify 'Ready to create New collection' pane appears.
20. Click the Finish button.
21. Verify a new collection 'Bahasa Malaysia Books' appears on the Collections view.

**Cleanup Steps**

1. Close Bloom application.
