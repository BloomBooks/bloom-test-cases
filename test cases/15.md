# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Open Notepad editor.
2. Open Report a Problem dialog from Bloom application.
3. Click Alt+Tab.
4. Verify the Notepad editor opens.
5. Minimize the Notepad editor.
6. Verify the error dialogue shouldn't disappear from view (should stay on top of Bloom)

**Cleanup Steps**

1. Close Bloom application.
