# Steps

**Setup Steps**

1. Download Turka Decodable Reader from the given link : https://drive.google.com/drive/folders/1WWDT2Iz7Q6Mh0Q_96Jiy1Y-RM-NG7UoS
2. Open it in the Bloom application.
3. Click the Settings button.
4. Verify the Settings dialog opens.
5. Click the Change link button under Local Language.
6. Enter Turka in the Search box.
7. Verify Turka language bar appears.
8. Click on the Turka language bar.
9. Click the OK button.
10. Click the Restart button.
11. Verify the Bloom gets restarted.

**Action Steps**

1. Click the EDIT THIS BOOK button.
2. Verify the book opens in the Edit view.
3. Click Page 1.
4. Verify the default font  of the Turka DR book should be there.
5. Verify the default font size of the Turka DR book should be there.
6. Click Page 2.
7. Verify the same font appears in Page 1 should be there.
8. Verify the same font size appears in Page 1 should be there.

**Cleanup Steps**

1. Delete the Turka DR book.
2. Close Bloom application.
