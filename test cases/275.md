# Steps

**Setup Steps**

1. Open Bloom from File Explorer C:\Users\Documents\Bloom\English Books
2. Launch Bloom application by double clicking on the “English Books(.Bloom Collection)”.

**Action Steps**

2. Click the  "Other Collection" from the Standard Toolbar.
3. Verify that the "Open/Create Collections" dialog box is displayed.
4. Click "Browse for another collection on this computer" from the dialog box.
5. Verify that the "Open Collection" dialog box is displayed.
6. Right Click on the "English Books" and rename it to "English Books 1"
7. Click the "Open" button.
8. Verify that the "Folder In Use" dialog box is displayed with "The action can't be completed because the folder or file in it is open in another program" message on it.

**Cleanup Steps**

1. Remove the English Book Collection folder.
2. Close Bloom application.
