# Steps

**Setup Steps**

1. Download the template from the given link: https://bloomlibrary.org/my-books/book/3HWkAfOsVW
2. Verify the template book opens in the Bloom application.

**Action Steps**

1. Click the Basic book under Templates.
2. Click MAKE A BOOK USING THIS SOURCE button.
3. Verify the book opens in the Edit view.
4. Click the Add Page button.
5. Verify the Add Page dialog opens.
6. Select ‘Student SK’  under “FAL Bloom App Templates” heading.
7. Click the Add Page button.
8. Verify the page is added.
9. Click the Publish button.
10. Click the 'BloomPUB' button.
11. Verify 'Publish to BloomPUB' page opens.
12. Click the Preview button.
13. Verify the Book Grid appears on the page.
14. Verify all the book thumbnails appear on the Grid box.
15. Select any of the added books (eg., The Moon and the Cap)from the Grid box.
16. Verify the selected book appears in the Preview pane.

**Cleanup Steps**

1. Delete the template book.
2. Delete the Basic book.
3. Close Bloom application.
