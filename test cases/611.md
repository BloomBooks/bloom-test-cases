# Steps

**Setup Steps**

1. Launch Chrome browser.
2. Enter bloomlibrary.org in the browser window.
3. Verify the bloomlibrary.org opens in the browser window.

**Action Steps**

1. Click the 3 dots button which appears at the top right corner of the page.
2. Verify an option menu opens.
3. Click the 'Translate' option.
4. Verify 'Google Translate' dialog opens.
5. Click the 3 dots button in the dialog box.
6. Click the 'Choose another language' option.
7. Verify 'Language to translate into' dialog appears.
8. Click the language list box.
9. Verify the list box opens.
10. Select 'Portuguese' language.
11. Click the 'Translate' button.
12. Verify the page appears in Portuguese UI language with no errors.
13. Click the 'LER' tab which appears at the right side of the Bloom LOGO.
14. Verify Bloom library \ READ page opens.
15. Verify the page appears with Portuguese UI language no errors.

**Cleanup Steps**

1. Close bloomlibrary.org in Chrome browser.
2. Close the Chrome browser.
