# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Navigate to the Bloom Collection folder. (C:\Users\username\Documents\Bloom)
2. Right-click on the Bloom folder.
3. Verify an option menu appears.
4. Click the Properties option.
5. Verify Bloom Properties dialog opens.
6. Click the 'Share' button under Network Path.
7. Verify Network access dialog opens.
8. Click the dropdown list box.
9. Select 'Everyone'.
10. Click the Add button.
11. Click the Share button.
12. Verify 'Your folder is shared' dialog appears.
13. Verify the location appears in the dialog. (eg., '\\USERNAME\Users\username\Documents\Bloom' )
14. Click the Done button.
15. Click the Close button.
16. Open Downloads folder. (or some other folder)
17. Press Windows + R keys.
18. Verify the Run dialog appears.
19. Paste the location (path) (eg., \\USERNAME\Users\username\Documents\Bloom)
20. Verify the Bloom Collection folder opens.

**Cleanup Steps**

1. Close Bloom application.
