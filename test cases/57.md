# Steps

**Setup Steps**

1. Navigate to AppData/Local/SIL/Bloom in the computer.
2. Delete the folder(s) corresponding to Bloom installation.
3. Delete the Bloom folder.
4. Launch Bloom application. (eg., Bloom Alpha)

**Action Steps**

1. Verify a Startup Wizard appears on the screen.
2. Click the 'I accept the terms of the license agreement' button.
3. Verify the new Open/Create Collections dialog opens.
4. Verify a UI language box appears at the top right corner of the page.
5. Click the (English) UI language dropdown list box.
6. Select 'Tamil' UI language.
7. Verify the UI language is changed to Tamil.
8. Click 'புதிய தொகுப்பை உருவாக்கு '(Create New Collection)
9. Click 'அடுத்த ' (Next) button.
10. Enter 'Tamil' in the Search by name,code or country text box.
11. Select 'Tamil' language bar.
12. Select 'Tamil' Script box.
13. Click 'அடுத்த ' (Next) button.
14. Verify Font and Script page opens.
15. Click 'அடுத்த ' (Next) button.
16. Verify language and region page opens.
17. Click 'அடுத்த ' (Next) button.
18. Verify 'தமிழ் புத்தகங்கள்' appears in Collection Name box.
19. Click 'அடுத்த ' (Next) button.
20. Click 'பூர்த்தி' (Finish) button.
21. Verify a Tamil Book Collection (தமிழ் புத்தகங்கள்)  opens in the Bloom application.
22. Verify Tamil UI language appears on the Bloom application.
23. Verify Software Updates dialog appears.
24. Click the 'சரி'  (OK) button.
25. Click 'மற்ற தொகுப்பு' (Other Collection) button.
26. Verify திறக்க/உருவாக்க தொகுப்புகள்'  (Open/Create Collections) dialog opens.
27. Click 'புதிய தொகுப்பு உருவாக்க '(Create New Collection)
28. Click 'அடுத்த ' (Next) button.
29. Enter 'English' in the Search by name,code or country text box.
30. Select 'English' language bar.
31. Click 'அடுத்த ' (Next) button.
32. Verify Font and Script page opens.
33. Click 'அடுத்த ' (Next) button.
34. Verify language and region page opens.
35. Click 'அடுத்த ' (Next) button.
36. Verify 'English புத்தகங்கள்' appears in Collection Name box.
37. Click 'அடுத்த ' (Next) button.
38. Click 'பூர்த்தி' (Finish) button.
39. Verify an English Book Collection (English புத்தகங்கள்)  opens in the Bloom application.

**Cleanup Steps**

1. Delete the Tamil Book Collection.
2. Delete the English Book Collection.
3. Close Bloom application.
