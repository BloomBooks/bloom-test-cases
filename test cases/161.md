# Steps

**Setup Steps**

1. Launch Bloom application.
2. Verify 7-zip is installed in the system.

**Action Steps**

1. Click The Moon and the Cap Shell book.
2. Click MAKE A BOOK USING THIS SOURCE button.
3. Verify the book opens in the edit mode.
4. Click the Credits Page.
5. Click Needs Copyright button.
6. Verify Copyright and License dialog opens.
7. Enter 'Tester' in the Copyright Holder name box.
8. Click the OK button.
9. Click the Publish tab.
10. Click the BloomPUB button.
11. Verify Publish as BloomPUB view opens.
12. Click the Preview button.
13. Click Save BloomPUB file radio button.
14. Click the Save button.
15. Verify Save As dialog opens.
16. Click the Save button.
17. Verify the book is saved as .bloomPUB file.
18. Right-click the .bloompub file.
19. Verify an option menu opens.
20. Click 7-zip.
21. Verify the submenu opens.
22. Click Open Archive.
23. Click the Extract button.
24. Verify that all the supported file appear.
25. Verify only the whitelist files should appear. (eg., .htm/.css/.js/.png etc.,)

**Cleanup Steps**

1. Delete the Moon and the Cap book.
2. Close Bloom application.
