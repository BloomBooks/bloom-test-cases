# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Basic Book Template under Sources for New Book in the left pane.
2. Verify the MAKE A BOOK USING THIS SOURCE button displays in the right pane.
3. Click the MAKE A BOOK USING THIS SOURCE button.
4. Verify that the template opens in edit mode and the edit cursor is displayed in the Title box.
5. Type "Basic Book" on the Title.
6. Click on the Add Page button from the Edit Pane.
7. Verify the Add Page... dialog appears.
8. Click the 'Just Text' template.
9. Verify the 'Just Text' template selected.
10. Click the ADD PAGE button.
11. Verify that Page "1" is added to the left pane.
12. Verify that Page "1" is editable on the right pane.
13. Verify the "A5 Portrait" displays on the "Choose a page size and orientation" dropdown.
14. Click the "Choose a page size and orientation" dropdown.
15. Verify that the different page size options are displayed.
16. Click on the ' Device 16x9 Landscape' Option.
17. Verify the "Device 16x9 Landscape" displays on "Choose a page size and orientation" dropdown.
18. Verify the Page format is changed to "Device 16x9 Landscape mode".
19. Type the following sentence into Page "1":

20. Verify that the Danger triangle error message "This box has more text than will fit" does not appear.
21. Verify the vertical scroll bar appears along with the text.
22. Click the "Choose a page size and orientation" dropdown.
23. Verify that the different page size options are displayed.
24. Click "Device 16x9 Portrait" Option.
25. Verify the "Device 16x9 Portrait" displays on "Choose a page size and orientation" dropdown.
26. Verify the Page format is changed to "Device 16x9 Portrait".
27. Verify that the Danger triangle error message "This box has more text than will fit" does not appear.
28. Verify the vertical scroll bar appears along with the text.
29. Click on the End of the Sentence from Page 1.

**Cleanup Steps**

1. Delete the Basic Book.
2. Close the Bloom Application.
