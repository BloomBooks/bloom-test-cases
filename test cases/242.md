# Steps

**Setup Steps**

1. Download French Decodable Leveled 2024-05-07.BloomPack from the given link : https://drive.google.com/drive/folders/1WWDT2Iz7Q6Mh0Q_96Jiy1Y-RM-NG7UoS (Needs developer's approval)
2. Double click on French Decodable Leveled 2024-05-07.BloomPack file.
3. Verify Bloom Pack Installation dialog opens.
4. Click the OK button.
5. Verify the book opens in the Bloom application.

**Action Steps**

1. Click French DR book thumbnail under French Decodable Leveled heading (which appears at the bottom left corner of the Collections view)
2. Click the EDIT THIS BOOK button.
3. Verify Stage x of x appears.
4. Verify 'Loading..' appears for several seconds under Sample words in this page title.
5. Click on another page (eg., Credits Page) before 'Loading' disappears.
6. Verify that no yellow box error occurs.

**Cleanup Steps**

1. Delete the French Decodable Leveled Book.
2. Close Bloom application.
