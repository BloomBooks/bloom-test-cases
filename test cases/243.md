# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click on “Basic Book” under “Sources for New Books” on the left pane.
2. Verify the MAKE A BOOK USING THIS SOURCE button appears in the Right pane.
3. Click the MAKE A BOOK USING THIS SOURCE button.
4. Verify the Edit Mode appears.
5. Click on the Front Cover page.
6. Verify the Front Cover page displays on the Right pane.
7. Verify the Edit Cursor displays on the Front Cover page.
8. Type Basic Book Test.
9. Click on the Add page button from the Left pane.
10. Verify Add page dialog appears.
11. Click the 'Just a text' template.
12. Verify the 'Just a text' template preview appears.
12. Click the ADD PAGE button.
13. Verify the Page "1" appears on the Right pane to Edit.
14. **Cleanup Steps**

1. Delete the Basic Book Test.
2. Close Bloom application.
