# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Other Collection button.
2. Verify Open/Create Collections dialog opens.
3. Click 'Create New Collection' button.
4. Verify Create New Bloom Collection dialog opens.
5. Type 'te' in the Search bar.
6. Verify the 'Script and Variant' link button enables.
7. Click the 'Script and Variant' link button.
8. Verify the 'Script and Variant' dialog box opens.
9. Click 'Script' pull down menu.
10. Verify the Script pull down menu opens.
11. Click 'Zanabazar Square' script.
12. Click the 'Region' pull down menu.
13. Verify the 'Region' pull down menu opens.
14. Click 'World' Region.
15. Type 'qtc' in the Variant text box.
16. Click the OK button.
17. Verify a complex language code is created with a different regions, script and variant.
18. Click the Next button.
19. Verify Create New Bloom Collection dialog opens.
20. Click the Next button.
21. Verify Collection Name dialog opens.
22. Click the Next button.
23. Verify the Ready to create new collection dialog box opens.
24. Click the Finish button.
25. Verify a new book collection is created successfully.

**Cleanup Steps**

1. Close Bloom application
