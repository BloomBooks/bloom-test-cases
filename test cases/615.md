# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Collections tab.
2. Verify there is no light-colored line below the Collections tab button.
3. Verify there is no dark bar running across the bottom of the window.
4. Click the Basic Book thumbnail.
5. Click the MAKE A BOOK USING THIS SOURCE button.
6. Verify Bloom opens in the Edit tab.
7. Verify there is no light-colored line below the Edit tab button.
8. Verify there is no light-colored line below the "Pages" label on the left side of the window.
9. Verify there is no dark bar running across the bottom of the window.
10. Click the 3-bar button on the right of the window to open the tool panel.
11. Verify the More... button displays correctly at the bottom of the tool panel.
12. Click the Publish tab.
13. Verify there is no light-colored line below the Publish tab button.
14. Verify there is no dark bar running across the bottom of the window.

**Cleanup Steps**

1. Close Bloom application.
