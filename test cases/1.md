# Steps

**Setup Steps**

1. Download  Bloom Alpha build.
2. Type Language settings in the Windows Search bar.
3. Verify Language settings option appears.
4. Click Language settings option
5. Verify Language settings window opens.
6. Click the 'Add a language' button under Preferred languages.
7. Verify Choose a language to install dialog opens.
8. Enter "spanish mexico" in the Search bar.
9. Verify Spanish Mexico language appears.
10. Select Spanish Mexico language
11. Click the Next button.
12. Click the Install button.
13. Verify the installation completed.
14. Restart Windows.
15. Verify ‘ESP’ Spanish Mexico language appears as the windows system language.

**Action Steps**

1. Double click on the Bloom Alpha build.
2. Verify "Bloom Alpha has missing system dependencies' dialog.
3. Click the Install button.
4. Verify 'Installing ..' dialog appears.
5. Verify Bloom Alpha dialog appears with 'I accept terms of the..' button.
6. Verify Bloom Alpha installed successfully.
7. Verify Bloom Alpha (open/create collections) dialog appears with Spanish UI language.

**Cleanup Steps**

1. Uninstall the Spanish Mexican language.
2. Close Bloom application.
