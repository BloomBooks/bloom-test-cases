# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click a Basic Book under Templates in the Collections view.
2. Verify MAKE A BOOK USING THIS SOURCE button appears on the preview pane.
3. Click MAKE A BOOK USING THIS SOURCE button.
4. Verify the book opens in the Edit mode.
5. Click the Add Page button.
6. Verify the Add Page dialog opens.
7. Click the 'Just Text' template.
8. Click the Add Page button.
9. Verify the 'Just Text' is added.
10. Verify the page is added in the edit mode.
11. Hover the mouse over the text page.
12. Verify a gear icon appears at the bottom left corner of the page.
13. Click the gear icon.
14. Verify the Format dialog opens.
15. Drag the Format dialog to the right side of the window.
16. Verify the dialog should be moved.
17. Drag the Format dialog out of the Bloom Window.
18. Verify the format dialog should only be dragged within the window.
19. Click the zoom in (+) button to increase the zoom size to 120%
20. Verify the bottom of the page is out of the screen.
21. Scroll down the page.
22. Click the gear icon of the text page.
23. Verify the Format dialog appears.

**Cleanup Steps**

1. Delete the Test Book.
2. Close Bloom application.
