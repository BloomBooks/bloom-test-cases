# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Settings button.
2. Verify the Settings dialog opens.
3. Click the Languages tab.
4. Click the 'Change' button.
5. Verify the 'Choose Language' dialog opens.
6. Enter 'whatchamacallit' (which has more than 8 characters) on the search box.
7. Verify a newly created unlisted language 'qaa-x-whatchamallit' appears under the Unlisted language label.
8. Press Ctrl and the (i) button.
9. Verify the Unlisted Language Tag dialog opens.
10. Verify the BCP 47 Tag should only show the language name with eight characters. (eg., qaa-x-whatcham)
11. Press Ctrl and the (i) button.
12. Verify an user interface dialog opens.
13. Enter 'whatchamacallit' on the text box.
14. Click the OK button.
15. Verify a warning message "This is not in a valid IETF BCP 47 format:whatchamacallit" appears on the screen.

**Cleanup Steps**

1. Close Bloom application.
