# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Template Starter book under Specialize Templates heading.
2. Verify the MAKE A BOOK USING THIS SOURCE button.
3. Click the MAKE A BOOK USING THIS SOURCE button.
4. Verify the Book Opens in the Edit view.
5. Click the About page thumbnail.
6. Click on the 'My Template' name under Template Title.
7. Delete the 'My Template' name.
8. Enter 'Flower' as the Title name.
9. Click the Collections tab.
10. Click the EDIT THIS BOOK button.
11. Verify the book opens in the Edit view.
12. Click the Add Page button.
13. Verify the Add Page dialog opens.
14. Click 'Just a Picture' template.
15. Click the Add Page button.
16. Verify the Just a Picture page is added.
17. Click the Add Page button.
18. Verify the Add Page dialog opens.
19. Verify 'Flower' label name appears in the Add Page dialog.

**Clean-up Steps**

1. Delete the Template Starter
2. Close Bloom application.
