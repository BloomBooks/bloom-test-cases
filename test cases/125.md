# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click a Basic book.
2. Verify MAKE A BOOK USING THIS SOURCE button.
3. Click MAKE A BOOK USING THIS SOURCE button.
4. Verify the book opens in the edit mode.
5. Enter 'The Golden Touch' in the Front Cover (English) title box.
6. Click the 'Click to choose topic' link button.
7. Verify Choose Topic dialog opens.
8. Select 'Bible' (or any topic).
9. Verify the Bible topic is selected and appear in bold style(highlighted).
10. Select 'Primer' (or any topic).
11. Verify the Primer topic is selected and appear in bold style(highlighted).
12. Select 'Story Book' (or any topic).
13. Verify the Story Book topic is selected and appear in bold style(highlighted).

**Cleanup Steps**

1. Delete the Basic book.
2. Close Bloom application.
