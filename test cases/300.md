# Steps

**Setup Steps**

1. Download and Install Keyman Keyboard for Windows from the given link : https://downloads.keyman.com/windows/stable/18.0.243/keyman-18.0.243.exe
2. Download and Install UTB Unicode.kmx keyboard from the given link : https://drive.google.com/drive/u/0/folders/1a6McYeQA5MKyFvwlUcsxaAM-MfjCIj51
3. Launch Bloom application.

**Action Steps**

1. Click the Basic book under Templates.
2. Click MAKE A BOOK USING THIS SOURCE button.
3. Verify the book opens in the edit mode.
4. Click the Add Page button.
5. Verify Add page dialog opens.
6. Click 'Just a text' template.
7. Click the Add page button.
8. Verify the Just a text template is added.
9. Click the Keyboard icon from the taskbar.
10. Click the Keyman keyboard.
11. Switch to 'UTB Unicode' keyboard.
12. Long press 'Space bar' key. (or any character keys)
13. Verify the long press feature is disabled.
14. Type 'a in the text box.
15. Verify á should appear on the text box.
16. Type x'a
17. Verify áxá  should appear on the text box.
18. Switch to English Keyman keyboard.
19. Long press 'Space bar' key. (or any character keys)
20. Verify the long press feature is enabled.

**Cleanup Steps**

1. Uninstall UTB Unicode Keyboard.
2. Uninstall Keyman Keyboard.
3. Delete the Basic book.
4. Close Bloom application.
