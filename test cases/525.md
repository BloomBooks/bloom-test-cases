# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Moon and Cap Book.
2. Click the 'Make a book using this Source' button.
3. Verify the book open in the Edit mode.
4. Click the Add page button.
5. Verify the Add page dialog opens.
6. Click 'Just a Picture' template.
7. Click the Add page button.
8. Verify 'Just a Picture' template is added.
9. Click the Help (?) button from the Standard toolbar.
10. Verify the Help menu opens.
11. Click the 'Report a Problem' option.
12. Verify 'Report a Problem' dialog opens.
13. Enter “Testing DOKTC 525” into the “What were you doing?” text box.
14. Enter “tester@test.com” in the “Email Address” text box.
15. Click the Submit button.
16. Verify that an issue link should be created and appear on the dialog.
17. Click on the issue link.
18. Verify it opens in your default web browser.

**Cleanup Steps**

1. Delete the Moon and the Cap Book.
2. Close Bloom application.
3. In the browser, delete the issue.
