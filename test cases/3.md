# Steps

**Setup Steps**

1. Download the latest version of Bloom from bloomlibrary.org web site.

**Action Steps**

1. Right click on the BloomInstaller.exe file.
2. Verify a popup menu opens.
3. Click the 'Run as administrator' option.
4. Verify User Account Control dialog box appears.
5. Click the 'Yes' button.
6. Verify the Bloom application installed successfully.

**Cleanup Steps**

1. Delete the BloomInstaller file.
2. Uninstall Bloom application.
