# Steps

**Setup Steps**

1. Launch Bloom application to a collection with English as L1.

**Action Steps**

1. Click the Settings button.
2. Verify the Settings dialog box.
3. Click the Book Making tab.
4. Click the 'Special Script Settings' link button for the first language.
5. Verify 'Special Script Settings...' dialog box opens.
6. Click 'This script is written right to left like Arabic...' checkbox.
7. Click the OK button.
8. Click the Restart button.
9. Verify Bloom gets restarted.
10. Click the Settings button.
11. Verify the Settings dialog box.
12. Click the Book Making tab.
13. Click the ‘Special Script Settings’ link button for the first language.
14. Verify the ‘This script is written right to left like Arabic…’ checkbox is still in checked state.

**Cleanup Steps**

1. Uncheck the ‘This script is written right to left like Arabic…’ checkbox.
2. Click the Restart button.
3. Close Bloom application.
