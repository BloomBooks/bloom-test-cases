# Steps

**Setup Steps**

1. Launch Bloom application
2. Click the Settings button.
3. Verify the Settings dialog opens.
4. Click the Bloom Subscription tab.
5. Verify the Bloom Subscription tab opens.
6. Enter 'Test-727011-1339' in the Subscription code.
7. Verify the Restart button appears instead of the OK button.
8. Click the Restart button.
9. Verify Bloom gets restarted.

**Action Steps**

1. Click 'The Moon and the Cap' Shell book under Shell Books.
2. Click the 'MAKE A BOOK USING THIS SOURCE' button.
3. Verify the book opens in the Edit view.
4. Click 'A5 Portrait' page size orientation combo box. (from the Bloom toolbar)
5. Verify the list box opens.
6. Select 'Device 16x9 Landscape'option.
7. Verify the page orientation changes to landscape view.
8. Verify the Source bubble appears near the text box.
9. Click any text page. (eg., Page 6)
10. Verify the Source bubble is visible regardless of focus.
11. Click 'Device 16x9 Landscape' page size orientation combo box. (from the Bloom toolbar)
12. Verify the list box opens.
13. Select 'Device 16x9 Portrait'option.
14. Verify the page orientation changes to Portrait view.
15. Click any text page. (eg., Page 6)
16. Verify the Source bubble is visible regardless of focus.

**Cleanup Steps**

1. Delete the Subscription code.
2. Delete ‘The Moon and the Cap’ book.
3. Close Bloom application.
