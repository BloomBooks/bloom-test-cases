# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click The Moon and the Cap shell book.
2. Click MAKE A BOOK USING THIS SOURCE button.
3. Verify the book opens in the edit mode.
4. Click the Collections tab.
5. Verify the Collections view opens.
6. Click The Moon and the Cap shell book.
7. Click MAKE A BOOK USING THIS SOURCE button.
8. Verify the book opens in the edit mode.
9. Close Bloom application.
10. Restart Bloom application.
11. Verify that the Bloom application opens without crashing.

**Cleanup Steps**

1. Delete ‘The Moon and the Cap’ book.
2. Close Bloom application.
