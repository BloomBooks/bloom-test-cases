# Steps

**Setup Steps**

1. Press Windows + I keys.
2. Click the Time & Language button.
3. Verify the Settings dialog opens.
4. Click Language option (from the left pane)
5. Click Add a language (+) button.
6. Verify Choose a language to install dialog opens.
7. Enter 'Netherlands' in the search bar.
8. Select 'Dutch (Netherlands)' language.
9. Click the Next button.
10. Select 'Set as my Windows display language' under Install language pack .
11. Click the Install button.
12. Verify the Dutch language is installed.
13. Verify a warning message 'Setting Windows display language' appears in the dialog.
14. Click the 'No, Sign out later' button.
15. Restart Windows OS.

**Action Steps**

1. Download the older version of BloomInstaller.exe (eg., BloomInstaler.5.6.516) from blorg.
2. Double click on the BloomInstaller.exe file.
3. Verify 'Choose a Language' dialog appears.
4. Verify a warning message 'Our apologies, this program has not yet been localized for Dutch (Nederlands). Please, choose from one of the following languages..' appears in the dialog.
5. Click the OK button.
6. Verify that the Bloom was successfully installed in the system.

**Cleanup Steps**

1. Remove the Dutch (Netherlands) from system language.
2. Close Bloom application.
