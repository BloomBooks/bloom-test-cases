# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click on the "Basic Book" Template from the Sources For New Book pane.
2. Verify that the "Basic Book" preview appears on the Right side pane.
3. Click the Settings button in the header
4. Verify the Settings dialog opens
5. Click the "Change..." button under the Local Language
6. Verify that the 'Choose Language' dialog opens
7. Type 'tasm' in the Search box
8. Verify that the "Create Unlisted Language" button appears at the bottom-left corner of the dialog
9. Click the Create Unlisted Language button
10. Verify that the "Unlisted Language Tag" dialog opens
11. Click the Country drop down
12. Type the text "India"
13. Verify that a list of options appears based on the typed text
14. Select the option "India"
15. Click the OK button in the "Unlisted Language Tag" dialog
16. Verify that the "Unlisted Language Tag" dialog closes
17. Verify no error message appears on the screen
18. Click the OK button in the 'Choose Language' dialog
19. Verify that the 'Local Language' updated with the typed text 'tasm'

**Cleanup Steps**

1. Delete the Basic book.
2. Close Bloom application.
