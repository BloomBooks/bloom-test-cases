# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Close button.
2. Verify Bloom application closes.
3. Open a browser window.
4. Enter ‘bloomlibrary.org’ in the search bar.
5. Press the Enter key.
6. Verify the bloomlibrary web page opens.
7. Click the 'English' book container.
8. Verify English books web page opens.
9. Click any one of the book thumbnails. (eg., Wild Animals)
10. Verify the book (eg., Wild Animals)opens on a web page.
11. Click the 'Translate into your language' - Download into Bloom Editor' button.
12. Verify 'This site is trying to open Bloom' message appears with the Open button.
13. Click the Open button.
14. Verify ‘Here, it comes’ dialog appears.
15. Verify Downloading book progress dialog appears.
16. Verify Download complete dialog appears with OK button.
17. Click the OK button.
18. Verify the Download complete dialog closes.
19. Click the OK button.
20. Verify the book opens in the Bloom application.
21. Verify MAKE A BOOK USING THIS SOURCE button appears on the right pane.
22. Click on the MAKE A BOOK USING THIS SOURCE button.
23. Verify the book opens in the edit view.

**Cleanup Steps**

1. Delete the new book (eg.,Wild Animals).
2. Close Bloom application.
