# Steps

**Setup Steps**

1. Open Chrome browser. (or any browser)

**Action Steps**

1. Click the Settings option.
2. Verify the Settings dialog opens.
3. Click the Languages option.
4. Click Add Languages under 'Preferred languages'
5. Verify Add languages dialog opens.
6. Enter 'French' in the search box.
7. Select 'French-Francais' language.
8. Click the Add button.
9. Verify the languages are added.
10. Click the ellipse button near the French language.
11. Verify a popup menu opens.
12. Select 'Display Chrome in this language'
13. Click the Relaunch button.
14. Verify Chrome browser gets restarted.
15. Verify the UI language of Chrome appears in French.
16. Enter 'alpha.bloomlibrary.org' in the Search bar.
17. Press the Enter key.
18. Verify 'Loading translations..' appears on the page.
19. Verify the bloom library UI language is changed to French.

**Cleanup Steps**

1. Close the Chrome browser.
