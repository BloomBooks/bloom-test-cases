# Steps

**Setup Steps**

1. Download some images (which have some symbols in their names) from the given link: https://drive.google.com/drive/folders/1WM78n2636XtbT3Tr1Rx8LM0MQlsoa9hP (Needs Developer’s approval) to the Downloads folder.
2. Replace part of the picture name with the '+' symbol. (eg., 03_03+16x9.png)
3. Launch Bloom application.

**Action Steps**

1. Click an eBook under Templates in the Collections view.
2. Verify MAKE A BOOK USING THIS SOURCE button appears on the preview pane.
3. Click MAKE A BOOK USING THIS SOURCE button.
4. Verify the book opens in the Edit mode.
5. Enter 'Test Book' on the Front cover title box.
6. Click the Change Image button.
7. Verify the Image toolbox opens.
8. Click the ‘File’ button.
9. Verify Choose Picture File dialog opens.
10. Click the Downloads option from the left pane.
11. Verify the Downloads folder opens.
12. Select the downloaded image which has a symbol(+) in its name.
13. Click the Open button.
14. Verify the picture appears in the preview pane.
15. Click the OK button.
16. Verify the picture appears in the Front cover page.

**Cleanup Steps**

1. Delete the images from the Downloads folder.
2. Delete the eBook.
3. Close Bloom application.
