# Steps

**Setup Steps**

1. Download the Decodable Reader from https://s3.amazonaws.com/bloom-problem-books/BL-10483.zip
2. Launch Bloom application

**Action Steps**

1. Click BL-10483 Decodable Reader Book.
2. Verify 'Make a Book using this Source' button appears.
3. Click the 'Make a Book using this Source' button.
4. Verify it opens in the edit mode.
5. Click the Add page button.
6. Verify the Add page dialog opens.
7. Click the 'Just text' template.
8. Verify the Just text page1 is added.
9. Enter some text like 'The Greek king Midas did a good deed for a Satyr. This prompted Dionysus, the god of wine, to grant him a wish. Midas asked for everything he touched to turn to gold. Dionysusâ€™ warned him not to do so, but Midas could not be swayed.' in the text page.
10. Click the Help button from the Help toolbar.
11. Click the 'Report a Problem' option.
12. Verify Report a Problem dialog opens.
13. Click the Learn More link button
14. Verify the error report appears on the page.
15. Verify the error report should show the subdirectory structure for 'Allowed Words' and 'Sample Texts' for Decodable Readers.

**Cleanup Steps**

1. Delete BL-10483 Decodable Reader book.
2. Close Bloom application.
