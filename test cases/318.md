# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Decodable Reader book.
2. Verify 'MAKE A BOOK USING THIS SOURCE' button appears.
3. Click the 'MAKE A BOOK USING THIS SOURCE' button.
4. Verify it opens in the edit mode.
5. Click the Add page button.
6. Verify the Add page dialog opens.
7. Select 'Just text' template.
8. Click the Add page button.
9. Verify the Just text template is added.
10. Type "aa bd a#a" in the text box.
11. Verify all the letters including the 'hash' (which appears in between the words) should be highlighted.
12. Press the Space key.
13. Enter '#' again.
14. Verify the Special character is not highlighted.
15. Press the Enter key.
16. Type "The a---s, d***, and ### aren't lost." in the text page.
17. Verify the special character which appears in between the words should be highlighted.
18. Verify the special characters which appear after the word should not be highlighted.

**Cleanup Steps**

1. Delete the Decodable Reader book.
2. Close Bloom application.
