# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click Basic Book in the Templates section under
2. Sources for New Books on the left pane.Verify the MAKE 3. A BOOK USING THIS SOURCE button appears in the right pane.
4. Click the MAKE A BOOK USING THIS SOURCE button.
5. Verify the Edit Mode appears.
6. Click on the Front Cover page.
7. Verify the Front Cover page displays on the right pane.
8. Verify the Edit Cursor displays on the Front Cover page.
9. Type Basic Book Test.
10. Click on the Add Page button from the left pane.
11. Verify Add Page dialog appears.
12. Click the Just a text template.
13. Verify the Just a text template preview appears.
14. Click the ADD PAGE button.
15. Verify the Page “1” is added on the left pane.
16. Verify the Page “1” appears on the right pane to edit.
17. Open the Notepad application.
18. Type: <div>blah&</notdiv>
19. Press Ctrl+A to select all.
20. Press Ctrl+C to copy.
21. Switch to the Bloom Application.
22. Click on Page “1”.
23. Press Ctrl+V to paste.
24. Verify it should just show the raw content that the user pasted.

**Cleanup Steps**

1. Delete the Basic Book Test.
2. Close Bloom application.
