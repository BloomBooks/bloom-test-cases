# Steps

**Setup Steps**

1. Rename c:\programdata\sil\Art Of reading (for AOR 3.2) or c:\programdata\sil\ImageCollections (for subsequent AORs).
2. Launch Bloom collection.

**Action Steps**

1. Click a Basic book.
2. Verify MAKE A BOOK USING THIS SOURCE button.
3. Click MAKE A BOOK USING THIS SOURCE button.
4. Verify the book opens in the edit mode.
5. Click the 'Change Image' button on the picture pane.
6. Verify the Image Toolbox dialog opens.
7. Click the Image Galleries button.
8. Verify a message "this computer doesn't appear to have any galleries installed yet.”  with ‘Download Art of Reading Installer’ hyperlink button.
9. Click on the hyperlink.
10. Verify it opens the AOR installer page. (eg., https://bloomlibrary.org/page/resources/art-of-reading)

**Cleanup Steps**

1. Delete the Basic book.
2. Close Bloom application.
