# Steps

**Setup Steps**

1. Launch Bloom application
2. Click the Settings button.
3. Verify the Settings dialog opens.
4. Click the Bloom Subscription tab.
5. Verify the Bloom Subscription tab opens.
6. Enter 'Test-727011-1339' in the Subscription code.
7. Verify the Restart button appears instead of the OK button.
8. Click the Restart button.
9. Verify Bloom gets restarted.

**Action Steps**

1. Click a Digital Comic book under Specialized Templates in Collections tab.
2. Verify MAKE A BOOK USING THIS SOURCE button.
3. Click MAKE A BOOK USING THIS SOURCE button.
4. Verify the book opens in the edit mode.
5. Verify the Overlay Tool menu automatically opens on the rightmost pane.
6. Click the Collections tab.
7. Click a Paper Comic book under the Specialized Templates in Collections tab.
8. Verify MAKE A BOOK USING THIS SOURCE button.
9. Click MAKE A BOOK USING THIS SOURCE button.
10. Verify the book opens in the edit mode.
11. Verify the Overlay Tool menu automatically opens on the rightmost pane.

**Cleanup Steps**

1. Delete the Subscription code.
2. Delete the Digital Comic book.
3. Delete the Paper Comic book.
4. Close Bloom application.
