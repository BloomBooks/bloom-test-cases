# Steps

**Setup Steps**

1. Delete this folder: AppData\Local\SIL\Bloom\Collections\Books From BloomLibrary.org\Story Primer.
2. Launch Bloom application

**Action Steps**

1. Download the Story Primer template book from the given link : https://bloomlibrary.org/resources/create-templates/book/zpHSUeny47
2. Verify the template book opens in the Bloom application.
3. Verify MAKE A BOOK USING THIS SOURCE button appears on the preview pane.
4. Click MAKE A BOOK USING THIS SOURCE button.
5. Verify the book opens in the Edit mode.
6. Click the Add page button.
7. Verify Add page dialog opens.
8. Verify all the Story primer thumbnails should show up.
9. Verify they are unique
10. Click on any of the Story primer thumbnails.
11. Verify it should not be slow.
12. Select Full Page thumbnail under Story Primer.
13. Click the Add Page button.
14. Verify 'localhost:8089 says 'applying bloom-resizable to a bloom-canvas may not work..' with an OK button.
15. Click the OK button.
16. Verify the page is added in the Edit view.

**Cleanup Steps**

1. Delete the Story Primer book.
2. Close Bloom application.
