# Steps

**Setup Steps**

1. Launch Bloom application.

**Action Steps**

1. Click the Other Collection button.
2. Verify Open/Create Collections dialog opens.
3. Click the 'Create New Collection' button.
4. Verify Create New Bloom Collection dialog opens.
5. Click the 'Can't find your language?' link button appears.
6. Verify About The ISO Language Registry dialog opens.
7. Click 'About Language 639-3 Codes & How to Apply For New Ones' link button.
8. Verify a corresponding web page opens.
9. Click the OK button.
10. Verify an unlisted language appears on the language pane.
11. Verify custom code 'qaa' appears near Script and Variant link button.
12. Click the script and Variant link button.
13. Type 'qtz' in the variant text box.
14. Click the OK button.
15. Verify the custom code changes to 'qtz' instead of 'qaa'.

**Cleanup Steps**

1. Close Bloom application
