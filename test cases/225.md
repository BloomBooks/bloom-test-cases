# Steps

**Setup Steps**

1. Launch Bloom application

**Action Steps**

1. Click a basic book from the left pane.
2. ) Verify “MAKE A BOOK USING THIS SOURCE” BUTTON gets displayed on the right pane.
2. Click the “MAKE A BOOK USING THIS SOURCE” button.
3. Verify that the book is displayed in the “Edit” view.
4. Click the “?” dropdown menu button from the top right side of the standard toolbar.
5. Click the “Release Notes (Web)” from the menu displayed.
6. Verify that the “Release Notes” Page opens in a browser with the following site address  “https://docs.bloomlibrary.org/Release-Notes/”.

**Cleanup Steps**

1. Close Bloom application.
2. Close “Release-Notes” web browser.
