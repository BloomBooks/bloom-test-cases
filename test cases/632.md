# Steps

**Setup Steps**

1. Double click on the Bloom folder. (eg., C:\Users\UserNamei\Documents\Bloom)
2. Verify the bloom folder opens.
3. Right click on .Bloom Book Collection.
4. Verify a popup menu opens.
5. Select the 'Read-only' checkbox.
6. Select the 'Hidden' checkbox.
7. Click the OK button.
8. Double click on .Bloom Book Collection.
9. Verify the book opens in the Bloom application.

**Action Steps**

1. Click the Settings button.
2. Verify the Settings dialog opens.
3. Click the 'Change' link button under Language 2 (National Language)
4. Select 'En'  language bar.
5. Click the OK button.
6. Click the Restart button.
7. Verify Fatal dialog with docs.Blorg link displays at top of error message in fatal error dialog.

**Cleanup Steps**

1. Close Bloom application.
