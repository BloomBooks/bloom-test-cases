# Steps

**Setup Steps**

1. Delete Bloom folder from C:\Users\UserName\AppData\Local\SIL
2. Delete Bloom folder from C:\Users\Siltest\AppData\Local
3. The computer has at least one administrator account and a standard (non-admin) user account.
4. An older version of the Bloom application is available.

**Action Steps**

1. Log in as administrator.
2. Verify the windows OS opens in the administrator account.
3. Install Older version of Bloom.
4. Verify the installation complete successfully with no errors.
5. Log out from the Administrator account.
6. Log in as a standard user (non-admin) in Windows OS.
7. Launch Bloom application.
8. Verify Bloom shows License terms dialog.
9. Accept the license terms.
10. Verify Bloom proceeds to open normally.
11. Verify no dialog should appear offering a choice of update options.
12. Click the Get Help(?) menu from the toolbar.
13. Verify the Option menu opens.
14. Click the 'Check for New Version' option.
15. Verify a message appears "Your system administrator manages Bloom updates for this computer"

**Cleanup Steps**

1. Close Bloom application.
